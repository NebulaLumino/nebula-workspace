# Quick Start Guide

## Running Commands

```bash
# Check status
openclaw status

# View logs
openclaw logs

# Run heartbeat
openclaw heartbeat
```

## Available Skills

- `gmail` - Email management
- `github` - GitHub operations  
- `things-mac` - Things 3 integration
- `weather` - Weather forecasts
- `playwright` - Browser automation
- `volcengine-seedream` - AI image generation
- `volcengine_seedance` - AI video generation

## File Locations

- Workspace: `~/.openclaw/workspace/`
- Skills: `~/.openclaw/workspace/skills/`
- Tasks: `~/.openclaw/workspace/tasks/`
- Memory: `~/.openclaw/workspace/memory/`
