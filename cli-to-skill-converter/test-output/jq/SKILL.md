## Options

- `--null-input`: use `null` as the single input value;
- `--slurp`: the single input value;
- `--compact-output`: compact instead of pretty-printed output;
- `--raw-output`: output strings without escapes and quotes;
- `--raw-output0`: implies -r and output NUL after each output;
- `--join-output`: each output;
- `--ascii-output`: using escape sequences;
- `--tab`: use tabs for indentation;
- `--indent`: n            use n spaces for indentation (max 7 spaces);
- `--unbuffered`: flush output stream after each output;
- `--stream`: parse the input value in streaming fashion;
- `--stream-errors`: an array;
- `--seq`: parse input/output as application/json-seq;
- `--from-file`: file      load filter from the file;
- `--arg`: name value      set $name to the string value;
- `--argjson`: name value  set $name to the JSON value;
- `--slurpfile`: from the file;
- `--rawfile`: name file   set $name to string contents of file;
- `--args`: string values;
- `--jsonargs`: JSON values;
- `--exit-status`: set exit status code based on the output;
- `--build-configuration`: show jq's build configuration;
- `--help`: show the help;
