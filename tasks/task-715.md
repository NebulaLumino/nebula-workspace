## App Concept

Generate inclusive, compelling job descriptions that attract the right candidates. Input: job title, department, seniority level, remote policy, key responsibilities, required qualifications, nice-to-have skills, salary range (optional), benefits highlights. Output: full job description with engaging intro paragraph, responsibilities as action-oriented bullets, requirements clearly separated (required vs. preferred), salary/compensation section, benefits overview, and application CTA. Includes inclusive language review.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
