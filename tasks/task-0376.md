## App Concept

A Next.js 16 web app that generates structured Quarterly Business Review deck outlines with wins, losses, pipeline analysis, forecast, and team commitments. Designed for sales, exec, and cross-functional reviews.

## Features

- Quarter context: quarter name, team, company stage
- Wins section: top 3-5 deals, revenue closed, avg deal size
- Losses section: top losses, loss reasons, competitive takeaways
- Pipeline coverage: current pipeline, coverage ratio, gaps
- Forecast accuracy: committed vs achieved
- Team performance: top performers, coaching opportunities
- Product feedback themes
- Next quarter commitments and focus areas

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
