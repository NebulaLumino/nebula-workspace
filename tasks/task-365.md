## App Concept

A Next.js 16 web app that helps founders and analysts estimate market size using TAM/SAM/SOM framework. Users input industry, geography, customer segment, and pricing; the AI calculates and justifies market size estimates with supporting data and methodology.

## Features

- Industry and geography selection
- Customer segment definition (B2B, B2C, B2B2C)
- Pricing and volume inputs
- TAM (Top-Down/Bottom-Up calculation)
- SAM: Serviceable Addressable Market with geographic/product constraints
- SOM: Serviceable Obtainable Market with market share assumptions
- Methodology explanation for each layer
- Comparable company analysis for validation
- Market growth rate projection (CAGR)
- Investor-ready chart visualization (bar chart: TAM/SAM/SOM)
- Export to PDF with charts

## Tech Stack

- Next.js 16 (App Router)  
- Tailwind CSS  
- DeepSeek API (OpenAI-compatible)  
- TypeScript
