## Overview
Build an AI-powered web app that generates API migration guides and deprecation notices for SDK/library upgrades. Users describe what changed in an API (e.g., "endpoint /users/{id}/profile was moved to /users/{id} and the response schema changed"), and the AI generates deprecation notices, migration instructions, changelog entries, and SDK update patches for multiple languages.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Describe API change → generate migration guide
- Support for REST, GraphQL, WebSocket, gRPC API changes
- Multi-language SDK examples: Python, JavaScript/TypeScript, Go, Ruby, Java, PHP, cURL
- Response schema diff: before/after JSON with field-level changes highlighted
- Migration urgency labeling: critical, high, medium
- Sunset date calculator with timeline (when to remove old endpoint)
- Code samples: working old code → working new code
- Automated codemod suggestion (jscodeshift, ts-migrate patterns)
- Deprecation header/response guidance (Deprecation, Sunset, Link headers)
- SDK version bump recommendation
- Consumer notification email/newsletter draft
- API changelog entry generation
- Feature flag recommendation for gradual rollout
