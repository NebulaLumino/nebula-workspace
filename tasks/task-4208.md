## Overview
Build an AI-powered web app that calculates statistically sound alert thresholds for business KPIs. Users input historical KPI data (time series), and the app calculates adaptive thresholds (upper/lower bounds) using standard deviation, rolling averages, or seasonal decomposition, then generates plain-English alert rules.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- OpenAI SDK

## Features
- Paste or upload historical KPI time series data
- Calculates thresholds using: standard deviation bands, rolling mean ±3σ, seasonal decomposition
- Visualizes the KPI with upper/lower threshold bands
- AI generates: "Alert when X happens because Y" rule descriptions
- Configurable sensitivity (how many σ from mean triggers alert)
- Supports weekly seasonality option
- Shows historical alert accuracy (% of past alerts that correlated with actual issues)
- Export alert rules as JSON, YAML, or plain text
