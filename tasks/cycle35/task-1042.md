## Concept & Vision
A tool for people who want to become confident public speakers — addressing both the practical skills and the anxiety that underlies the fear. The app generates a personalized speech, practice schedule, and anxiety management toolkit. Feels like having a speech coach and therapist in one.

## Specific Features
- Speech outline and full-text generator
- Audience analysis and message alignment
- Anxiety profile assessment
- Grounding techniques toolkit generator
- Rehearsal schedule builder
- Visual aid recommendations
- Q&A anticipation and response practice
- Post-speech debrief template

## Input Fields
- Occasion type (interview / wedding toast / work presentation / conference / classroom)
- Audience size and type
- Time limit
- Key message to convey
- Whether content is provided or needs to be generated
- Current anxiety level (1-10)
- Past speaking experience

## Output
Public speaking package:
- Full speech or outline
- Audience adaptation notes
- Anxiety management toolkit (breathing, grounding, cognitive reframing)
- Rehearsal schedule (3-5 sessions)
- Practice delivery notes
- Visual aid suggestions
- Q&A preparation brief
- Day-of anxiety protocol
- Post-event reflection template

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI, DeepSeek API.
