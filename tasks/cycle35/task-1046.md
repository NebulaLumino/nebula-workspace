## Concept & Vision
A tool for people navigating grief — whether from death, divorce, job loss, infertility, or other significant losses. The app generates a personalized grief support plan with evidence-based frameworks (Kübler-Ross, Dual Process Model, Meaning Reconstruction), rituals, and resources. Feels like compassionate guidance from someone who understands grief is not linear.

## Specific Features
- Grief type identifier (what kind of loss)
- Dual Process Model framework (loss-oriented vs. restoration-oriented coping)
- Kübler-Ross stage awareness (not linear — just a map)
- Meaning-making exercise generator
- Ritual and memorial creation tool
- Anniversary and milestone support planner
- "New normal" navigation guide
- When to seek professional help indicator

## Input Fields
- Type of loss (death of person / divorce / job loss / miscarriage / health loss / other)
- Relationship to the loss
- Time since loss
- Current symptoms and challenges
- Cultural or religious context
- Support system available
- What has helped and what hasn't
- Whether professional support is already in place

## Output
Grief support package:
- Loss acknowledgment statement
- Grief type and framework guide
- Dual Process Model navigation tips (oscillating between loss and restoration)
- Meaning-making exercises
- Ritual ideas (personalized to the relationship)
- Anniversary and milestone preparation plan
- "New normal" navigation guide
- Warning signs requiring professional support
- Resource recommendations (books, communities, professionals)
- Ongoing check-in questions

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI, DeepSeek API.
