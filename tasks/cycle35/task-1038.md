## Concept & Vision
A tool for ambitious professionals who set goals but struggle to execute them. The app takes a big goal and systematically breaks it down into quarterly, monthly, weekly, and daily action levels — with accountability checkpoints and contingency planning. Feels like a strategic planning session with yourself.

## Specific Features
- Annual goal to quarterly milestone translator
- Quarterly theme and focus generator
- Monthly priority selector
- Weekly planning generator (MITs — most important things)
- Daily time blocking assistant
- Project decomposition into tasks
- Buffer and contingency planning
- Progress review template generator

## Input Fields
- Annual goal(s) (1-3 big goals)
- Time horizon (current quarter / full year)
- Available hours per week for goal work
- Conflicting priorities or obligations
- Typical productivity patterns (morning person / evening / scattered)
- Accountability structure (solo / partner / team)

## Output
Quarterly planning package:
- Annual goal breakdown into quarterly milestones
- Current quarter theme and 3 major priorities
- Monthly breakdown for each priority
- Weekly planning template (MITs + tasks)
- Daily time-blocking suggestions
- Project task decomposition
- Buffer time allocation
- Bi-weekly check-in questions
- End-of-quarter review template

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI, DeepSeek API.
