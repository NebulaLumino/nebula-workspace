# Task 1036: AI Relationship & Communication Skills Training Generator

**Status:** ✅ Complete
**Priority:** HIGH
**Cycle:** 35 (1031-1060)
**Theme:** Human Potential, Philosophy & Future Studies

---

## Concept & Vision
A tool for anyone looking to improve their relationships — romantic, professional, familial, or social. The app generates personalized communication skill training plans and conversation scripts grounded in evidence-based approaches (Nonviolent Communication, Imago Therapy, emotional intelligence research). Feels like a compassionate communication coach.

## Specific Features
- Communication style assessment framework
- Nonviolent Communication (NVC) practice generator
- Difficult conversation script builder
- Active listening exercise generator
- Emotional trigger identification and response planning
- Conflict reframing tool
- Apology and repair conversation generator
- Relationship check-in template generator

## Input Fields
- Relationship type (romantic / family / professional / friend / ex-partner)
- Specific challenge or situation
- Communication style self-assessment
- What typically goes wrong in difficult conversations
-对方的 known personality traits and communication style
- Desired outcome
- Whether this is for ongoing improvement or a specific upcoming conversation

## Output
Communication skills package:
- Communication style profile
- Scenario-specific practice exercises (3-5 dialogues)
- NVC observation + feeling + need + request framework
- Anticipated responses and how to handle them
- Post-conversation reflection template
- Relationship check-in agenda template
- 30-day communication skill practice plan

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI, DeepSeek API.
