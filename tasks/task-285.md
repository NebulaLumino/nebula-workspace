# Task 285: AI Demo Request Follow-up Sequence Generator

**Status:** ✅ Completed — 2026-03-20
**Priority:** MEDIUM
**Category:** Sales Automation
**URL:** https://ai-demo-followup.vercel.app

## What It Does

Generates a complete multi-channel B2B demo follow-up sequence covering email, phone, LinkedIn, SMS, and a full demo-to-close playbook.

## Features

- 6-email pre-demo follow-up sequence with subject lines and body copy
- Phone/voicemail scripts (3 variants)
- LinkedIn outreach sequence (connection note + 2 messages)
- SMS sequence (3-touch text follow-up)
- Objection handling for 6 common demo objections
- Demo booking optimization + pre-demo email
- Post-demo 5-email close sequence
- Metrics to track (show rate, demo-to-pipeline, close rate)

## Tech

- Next.js 16, TypeScript, Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- Deployed on Vercel
