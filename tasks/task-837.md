## App Concept
Generate scenarios and explanations for cap table situations. Input: current ownership breakdown, new investment amount, new valuation, option pool size, existing investor rights. Output: cap table scenarios with dilution analysis, per-share price, exit proceeds at various multiples, and board seat implications.
