## Overview
Build an AI-powered web app that generates comprehensive migration guides for library upgrades and framework migrations. Users select a "from → to" migration (e.g., React 17 → React 18, AngularJS → Angular, Python 2 → Python 3, lodash 3 → lodash 4), paste their current code, and the AI generates a detailed, file-by-file migration plan with before/after code examples and estimated effort.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Pre-defined migration paths: React, Vue, Angular, AngularJS, Django, Rails, Node.js, Python, npm packages, Kubernetes, Docker
- Custom migration pair input
- Code diff generation with before/after
- Breaking change categorization: critical, high, medium, low
- Automated refactoring suggestion with code patches
- Deprecation warning audit across codebase
- Risk assessment for each change
- Estimated effort scoring (quick fix, multi-file refactor, architectural change)
- Migration order recommendation (what to migrate first)
- Rollback strategy documentation
- Post-migration verification checklist
- Integration with codemods where applicable
