## Description
Build an AI-powered currency volatility and hedging strategy advisor. Track major currency pairs (USD, EUR, GBP, JPY, CNY, etc.), analyze historical volatility regimes, calculate optimal hedge ratios using Value-at-Risk (VaR) models, recommend hedging instruments (forwards, options, ETFs), and generate AI-powered FX risk reports for businesses and investors.

## Requirements
- Real-time FX rate tracking (20+ currency pairs)
- Historical volatility analysis (HV, GARCH)
- Correlation matrix across currency pairs
- VaR / CVaR risk calculator
- Optimal hedge ratio optimizer
- Forward rate calculator
- Options strategy recommendations (FX options)
- ETF-based hedging suggestions
- AI currency risk narrative
- FX exposure report generator

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- ExchangeRate-API / Frankfurter API
- OpenAI SDK
- Recharts
- Python backend for GARCH/VaR models
