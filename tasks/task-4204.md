## Overview
Build an AI-powered web app that analyzes A/B test results and determines statistical significance. Users enter control/variant data (visitors, conversions), and the AI runs significance tests, calculates effect size, power, and confidence intervals — then generates a plain-English interpretation and recommendation.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK

## Features
- Input: sample sizes, conversions for control and variant
- Calculates: conversion rates, relative uplift, z-statistic, p-value
- Statistical significance threshold selector (0.05, 0.01, 0.10)
- Confidence interval display
- One-tailed vs two-tailed test option
- Minimum detectable effect (MDE) calculator
- AI-generated plain-English summary and recommendation
- Sample size calculator (pre-test planning)
- Result history in localStorage
