## Description
Build a Next.js app that generates structured grant proposals with abstract, aims, methodology, budget justification, and impact statement.

## Implementation Notes
- Form inputs: project title, research area, budget range, funding agency, duration, team size
- Output: project abstract, statement of need, specific aims, methodology, expected outcomes, broader impacts, budget justification, management plan
- Uses DeepSeek API via OPENAI_API_KEY env var
- Dark gradient UI with amber/orange theme
