## Overview
Build an AI-powered web app that automatically cleans messy datasets and detects statistical outliers. Users upload CSV/Excel data or paste a dataset URL, and the AI analyzes column types, missing values, duplicates, formatting inconsistencies, and outlier scores — then generates a cleaned version plus a detailed cleaning report.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- CSV/XLSX file upload or paste data
- AI column type detection (numeric, categorical, date, text, email, URL)
- Missing value analysis with recommended imputation strategies
- Duplicate row detection
- Outlier detection using IQR, Z-score, and isolation forest logic
- One-click apply all cleaning transformations
- Download cleaned CSV
- Cleaning report with before/after stats
- Column-by-column formatting fixes (trim, case normalization, date parsing)
