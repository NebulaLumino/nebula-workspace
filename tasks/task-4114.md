## Description
An AI-powered tool that helps people process grief through structured writing exercises. Users can write unsent letters to deceased loved ones, process complicated grief emotions, and generate letters to employers/family explaining their situation. Includes grief stage awareness and crisis resource integration.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User selects the type of grief support letter needed
2. AI guides user through prompts to express feelings and memories
3. AI generates a compassionate, personalized letter draft
4. User can edit and save letters (never sent — private journaling)
5. Includes grief stage information and resources
