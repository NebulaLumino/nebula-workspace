## App Concept

Upload or paste a legal contract and ask natural-language questions to find relevant clauses instantly. Input: PDF/text contract, query like "what are the termination clauses?" or "show me the indemnification section." Output: highlighted clause excerpts with summary, section reference, and risk flag for unusual language.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
