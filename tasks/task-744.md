## App Concept

Generate structured meeting notes summaries from transcripts or notes. Input: meeting topic, attendees, raw meeting notes or transcript, key decisions made, action items mentioned. Output: formatted meeting summary with date/attendees header, decisions log, action items table (task, owner, due date), open questions, and next meeting agenda suggestions. Includes AIQ (action item + owner + due date) extraction.

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
