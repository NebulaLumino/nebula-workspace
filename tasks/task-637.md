## App Concept

Generate XML sitemaps and SEO-friendly site structure documentation. Input: website URL, page list or site structure description, content update frequency per section, importance priority per section, list of URLs to exclude (admin, duplicate, paginated). Output: complete XML sitemap.xml formatted content, sitemap index suggestion for large sites, robots.txt updates, and a visual site hierarchy diagram.
