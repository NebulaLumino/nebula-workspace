## Project
`ai-risk-planner`

## Description
Build an AI-powered risk management tool that identifies, assesses, and plans mitigations for project risks. Analyzes project plans, team structure, and historical data to surface hidden risks and recommend contingency actions.

## Requirements
- Project plan import (Jira, Asana, CSV, doc upload)
- AI risk identification from project parameters
- Risk scoring (probability × impact matrix)
- Category classification (technical, resource, external, schedule)
- Mitigation strategy generation with cost/benefit analysis
- Contingency plan auto-generation
- Risk register with owner assignment
- Monte Carlo schedule simulation
- Risk trend dashboard and escalation alerts

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Jira / Asana API
- Vercel AI SDK
