## App Concept

A Next.js 16 web app that generates one-page product specification sheets with feature lists, technical requirements, integration details, and pricing. Used by product teams and B2B companies to communicate product capabilities.

## Features

- Product overview: name, version, category, tagline
- Key features list (6-10 bullets)
- Technical requirements: OS, hardware, dependencies
- Integration options: APIs, webhooks, third-party tools
- Pricing tiers: feature breakdown by plan
- Use cases and target industries
- Performance/specs table
- Download/copy-ready format

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
