## App Concept

Generate GDPR/CCPA-compliant Data Processing Agreements for SaaS companies. Input: data controller company, data processor company, categories of personal data processed, processing purposes, sub-processor list, data breach notification timeline, data transfer mechanism (SCCs, adequacy decisions). Output: complete DPA document with processing scope, processor obligations, security measures, data subject rights facilitation, audit rights, breach notification procedures, and standard contractual clauses.
