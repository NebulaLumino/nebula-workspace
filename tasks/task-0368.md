## App Concept

A Next.js 16 web app that generates comprehensive go-to-market strategies including target market definition, pricing model, sales channels, messaging pillars, and a phased launch timeline. Perfect for pre-launch startups and new product releases.

## Features

- Target market input: ICP, TAM/SAM/SOM, buyer personas
- Pricing strategy: model type, price points, discount tiers
- Channel selection: direct, partner, PLG, outbound, inbound
- Messaging pillars (3-5 core messages)
- Launch timeline: pre-launch, launch week, 30/60/90 day phases
- Competitive differentiation map
- Success KPIs and metrics

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
