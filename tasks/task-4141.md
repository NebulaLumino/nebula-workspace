## Description
Build an AI-powered cryptocurrency portfolio rebalancing tool. The app connects to major exchanges (Binance, Coinbase, Kraken) via API, analyzes current holdings against target allocation strategies, and generates optimal rebalancing orders. Features include tax-aware rebalancing, fee minimization, and risk-adjusted portfolio optimization.

## Requirements
- Exchange API integrations (Binance, Coinbase, Kraken)
- Portfolio analytics dashboard
- Tax-loss harvesting integration
- Custom allocation strategies
- Gas/fee optimization
- Real-time price feeds
- Rebalancing order generation
- Historical performance tracking
