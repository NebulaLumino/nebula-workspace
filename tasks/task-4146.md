## Project
`ai-contract-auditor`

## Description
Build an AI-powered smart contract security auditing tool. Upload a Solidity/Vyper contract or provide an on-chain address, and receive automated vulnerability detection (reentrancy, overflow, access control), AI-generated severity scores, exploit scenario descriptions, and recommended fixes. Integrates with Slither, Mythril, and manual audit workflows.

## Requirements
- Solidity/Vyper code upload & parsing
- Automated static analysis (Slither, Mythril patterns)
- AI vulnerability severity scoring
- Exploit scenario descriptions
- Recommended fix suggestions
- On-chain contract verification (Etherscan, Sourcify)
- Audit report PDF export
- Historical vulnerability database lookup

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Slither (static analysis)
- OpenAI SDK
- Ethers.js / Viem
- Etherscan / Sourcify API
- PDF generation (jsPDF)
