## Overview
Build an AI-powered web app that analyzes proposed database schema changes and assesses their impact. Users describe their current schema (or paste DDL), describe the migration, and the AI generates a detailed impact analysis: affected tables, breaking changes, backward compatibility risks, and migration SQL scripts.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- CodeMirror or Shiki for SQL highlighting
- OpenAI SDK

## Features
- Paste current schema DDL (CREATE TABLE statements)
- Describe desired changes in natural language (e.g., "rename column X to Y", "add nullable Z")
- AI generates: impact report (affected queries, downstream dependencies, breaking changes)
- Suggests migration SQL with backward-compatible steps
- Rollback plan generation
- Highlights deprecated column references
- Compatibility warnings for ORM frameworks (TypeORM, Hibernate)
- Export: impact report as Markdown, migration SQL as .sql file
