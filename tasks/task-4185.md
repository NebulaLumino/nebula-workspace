## Description
Build an AI-powered knowledge base tool that analyzes support tickets, documents, and product specs to auto-generate FAQs and structured knowledge base articles with maintenance reminders.

## Requirements
- Support ticket/email import (Zendesk, Intercom, Freshdesk CSV)
- Product manual/document upload
- AI question extraction and FAQ grouping
- Structured article generation (problem → solution format)
- Search-optimized article titles and meta descriptions
- Internal linking suggestions
- Article quality scoring (completeness, clarity, SEO)
- Content freshness tracking and update reminders
- "Was this helpful?" tracking and feedback loop
- Export to Confluence, Notion, Zendesk, Help Center

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Zendesk / Intercom API
- Vercel AI SDK
