## App Concept

A Next.js 16 web app that generates structured 30-minute B2B discovery call scripts with timed sections, specific questions, transition lines, and red flag warnings. Helps sales reps stay on track and uncover real pain points.

## Features

- Call structure: agenda, opener, discovery, demo fit, close next steps
- Timed sections (5-min increments) with what to cover
- Specific discovery questions organized by topic
- Transition lines between sections ("Great, now let me ask you about...")
- Red flag warnings: phrases that signal bad fit or disinterest
- Note-taking prompts at each section
- Objection pre-empts during the call
- Post-call action items template

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
