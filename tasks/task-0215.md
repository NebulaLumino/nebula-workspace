## Research Findings — Model Distillation 2026

### What Is Model Distillation?
Training a small model to mimic a large model's behavior. The small model learns from the large model's "soft predictions" (probability distributions), not just hard labels.

### Distilled Models (Production-Ready)

| Model | Size | Original | Quality Loss | Best For |
|---|---|---|---|---|
| **Phi-4 3.8B** | 3.8B | GPT-4 class | ~10-15% | Fast edge inference |

### Distillation Techniques

| Technique | How It Works | Quality |
|---|---|---|
| **Knowledge Distillation** | Student learns from teacher's logits | Best |
| **RL Distillation** | Use RLHF from large model as reward signal | Good |
| **Quantization** | Compress weights (FP16 → INT8 → INT4) | Varies |
| **Pruning** | Remove redundant neurons | Good |
| **Speculative Decoding** | Small draft + large verifier | Transparent speedup |

### Ollama for Local Distilled Models

```bash
# Quantized versions (smaller + faster)
ollama run llama3.2:3b-q4_0  # 4-bit quantized
```
