## App Concept

Generate long-form LinkedIn articles and thought leadership posts. Input: topic, key message, author's professional background, target audience, article length (short post, 500 words, 1000+ words), CTA goal (follow, comment, DM, visit link). Output: complete article with headline, subheadings, opening hook, body paragraphs, data points placeholder, closing CTA, and optimal posting time recommendation.

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
