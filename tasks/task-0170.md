## Research Findings — Local AI Development 2026

### Why Local AI?

| Benefit | Impact |
|---------|--------|
| **Privacy** | Prompts never leave your machine |
| **No API costs** | One-time download, unlimited use |
| **Offline** | Works on planes, secure environments |
| **Low latency** | No network round-trip |
| **Full control** | Choose model, quantization, system prompt |

### Apple Silicon Advantage
M-series unified memory = CPU + GPU + Neural Engine share one high-bandwidth pool → no VRAM bottleneck:
- **30B Q4 models** → 32GB RAM Mac minimum
- **70B Q4/Q5 models** → 64GB+ RAM Mac
- M5 Pro/Max (March 2026): significant speed improvements for LLM inference

### Top Local LLM Tools

**Ollama** (developer choice)
- Single command: `ollama run llama3.2`
- 100+ models: GPT-OSS, DeepSeek V3.2, Qwen3, Llama 4, Gemma 3, Mistral
- Cross-platform: macOS, Windows, Linux
- Desktop app (July 2025)
- **OpenAI-compatible API** — drop-in for any app using OpenAI SDK
- Integrates with LangChain, CrewAI
- `ollama serve` → local API server on port 11434

**LM Studio** (GUI/non-technical)
- Polished GUI for model discovery + download
- Built-in chat + conversation history
- OpenAI-compatible API server
- Often faster than Ollama on non-dGPU machines (Vulkan offloading)

**Jan** (open alternative)
- Cross-platform, local-first
- API-compatible with OpenAI

### Best Models to Run Locally (2026)

| Model | Size | Best For | Hardware |
|-------|------|----------|----------|
| **Gemma 3 4B** | 4B | Fast tasks, on-device | 8GB RAM |
| **Mistral Small 3** | ~7B | General use, fast | 16GB RAM |
| **Mistral Small 4** | 119B | High quality | 64GB RAM |
| **Llama 3.2 11B Vision** | 11B | Vision + text | 24GB RAM |
| **Qwen2.5 7B** | 7B | Coding, reasoning | 16GB RAM |
| **DeepSeek V3 671B** | 671B MoE | Top quality, 37B active | 128GB+ RAM |

### Quantization (GGUF Format)
- **Q4_K_M**: Good balance — most popular
- **Q5_K_S**: Higher quality, more RAM
- **Q8_0**: Near-lossless, large files
- Smaller quantizations (Q2, Q3): fit in less RAM, lower quality
