## Description
Build a Next.js app that generates structured conference abstracts with background, methods, results, conclusions, word count, and keywords.

## Implementation Notes
- Form inputs: paper title, findings summary, methodology, conference name, target word count
- Output: structured abstract (IMRAD format), word count analysis, suggested keywords, elevator pitch, reviewer angle Q&A
- Uses DeepSeek API via OPENAI_API_KEY env var
- Dark gradient UI with cyan/blue theme
