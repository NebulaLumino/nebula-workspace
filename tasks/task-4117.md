## Description
An AI-powered journaling tool that helps users build self-esteem through guided prompts, cognitive behavioral exercises, and personalized affirmations. The AI analyzes journal entries to identify negative self-talk patterns and generates targeted affirmations to counteract them.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User writes journal entries using AI-guided prompts
2. AI analyzes text for negative self-talk and cognitive distortions
3. AI generates personalized affirmations based on identified patterns
4. User reviews and saves affirmations to a daily deck
5. Daily affirmation notifications with rotation
