## App Concept

Generate full product catalog descriptions for catalogs and marketplaces. Input: product line name, category, number of products, target marketplace (Amazon, Etsy, Shopify, Walmart), tone, key selling points. Output: structured catalog entries with titles, descriptions, keywords, and variation descriptions for each product in the line.

- DeepSeek API (OpenAI-compatible)
