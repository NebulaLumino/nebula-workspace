## Research Findings — AI Voice / TTS Agents 2026

### TTS Leaders

| Tool | Strength | Best Use |
|---|---|---|
| **ElevenLabs** | Highest quality, voice cloning, 30+ languages | Brand voice, character voices |
| **OpenAI Voice API** | 6 voices, integrated with GPT-4o | Chatbots, real-time apps |
| **Cartesia** | Ultra-low latency (290ms), voice cloning | Real-time conversations |
| **VALL-E** | 3-second clone, high quality | On-device TTS |

### Voice Agent Platforms

**Retell AI**
- Build voice AI agents that answer phone calls
- 3rd-gen voice agent with 160ms latency
- Real-time conversational AI over phone
- Built-in IVR (phone tree), SIP integration

**Bland AI**
- Outbound phone calling agent
- Human-like voice, multi-call campaigns
- Best for: sales outreach, appointment reminders

**Daily (formerly Daily.co)**
- Real-time audio/video infrastructure
- Powers voice agents built on top
- Global coverage, low latency

### Real-Time Voice Pipeline
Microphone → STT → LLM → TTS → Speaker, all in <500ms to feel natural.

### OpenAI Realtime API
WebSocket-based, 10 pre-built voices, GPT-4o-mini, tool calling, 2024 release. Best for in-app voice assistants.
