## App Concept

Generate user-friendly error pages (404, 500, maintenance) that retain visitors. Input: error type (404 page not found, 500 server error, maintenance mode, age-restriction blocked), brand name, primary navigation options, contact/support option. Output: engaging error page copy with friendly tone, clear explanation of what happened, suggested next steps, search bar copy if applicable, navigation shortcuts, and a touch of brand personality/humor. Includes redirect suggestion logic.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
