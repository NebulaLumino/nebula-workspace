## Description

Next.js app that generates comprehensive dark matter and cosmological simulation parameter documents. Includes:
- Dark matter particle candidate analysis (WIMPs, axions, sterile neutrinos, fuzzy DM, PBHs)
- N-body/hydro simulation parameters (GADGET-4, RAMSES, AREPO, FIRE)
- Direct/indirect detection constraints (XENON, LUX, Fermi-LAT)
- Halo mass function and structure formation
- Relic density and cosmological parameter outputs

## Implementation

- Framework: Next.js 16 (App Router)
- AI: DeepSeek API via OpenAI SDK (compatible endpoint)
- UI: Dark gradient (gray-900/gray-950), lime accent color, clean form + markdown output
- Features: Preset configurations, real-time form, markdown rendering with tables
- Environment: .env.local with lazy OPENAI_API_KEY init
