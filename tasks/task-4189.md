## Description
Build an AI-powered quarterly business review tool that connects to business data sources (CRM, analytics, finance) and generates executive-ready quarterly briefs with highlights, challenges, metrics, and forward-looking recommendations.

## Requirements
- Multi-source data connectors (CRM, Google Analytics, Stripe, Salesforce, spreadsheet)
- AI narrative generation: CEO/Board-level QBR format
- KPI snapshot dashboard with auto-generated commentary
- Quarter-over-quarter comparison
- Win/loss analysis from CRM
- Financial highlights extraction
- Team/headcount narrative
- Competitive landscape summary
- Next quarter priorities and goals
- Branded slide deck export (PDF, PPTX)

## Tech Stack

- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Google Analytics / Stripe API
- Vercel AI SDK
