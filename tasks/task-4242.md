## Overview
Build an AI-powered web app that explains cryptic programming errors in plain English and suggests fixes. Users paste error messages (from any language/framework), stack traces, or even screenshots of terminal output, and the AI provides a clear explanation, root cause analysis, and copy-pasteable fix.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Paste error message / stack trace → plain English explanation
- Multi-language: TypeScript, Python, Go, Rust, Ruby, Java, C#, PHP, Node.js, Django, Rails, Spring
- Screenshot OCR: paste screenshot of terminal → extract and explain
- Root cause analysis: why did this happen
- Fix suggestion with code examples
- Related common errors: "users who saw this also saw..."
- Severity classification: syntax, runtime, logical, environmental
- Link to relevant documentation (MDN, Python docs, official guides)
- "Why is my AI code broken?" specialized category for LLM-generated code
- Stack trace line highlighting: which frame is the actual cause
- Suggest debugging commands to run for further diagnosis
