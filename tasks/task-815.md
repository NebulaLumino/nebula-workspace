# Task 815: AI Upsell/Cross-sell Recommendation Generator

**Status:** ✅ Complete
**Type:** App Generator — E-commerce/Sales
**Priority:** HIGH
**Cycle:** 28 (791-820)

## App Concept
Generate upsell and cross-sell recommendations for e-commerce. Input: products in cart, customer purchase history, product catalog categories, margin per product, seasonality. Output: recommended bundle pairings with pricing strategy, cross-sell sequencing (checkout, post-purchase, email follow-up), copy for upsell prompts, and projected AOV increase.

## Tech Stack
- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
- GitHub repo: `ai-upsell-cross-sell`
