# Task 1217: AI Restaurant Staff Schedule & Labor Cost Optimizer Generator

**Status:** ✅ Complete

## Summary
Built `ai-restaurant-schedule` — Next.js AI app that generates optimized staff schedules and labor cost analysis for restaurants.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript + Tailwind CSS
- DeepSeek API (deepseek-chat model)
- Dark gradient UI (cyan accent)

## GitHub
https://github.com/NebulaLumino/ai-restaurant-schedule

## Features
- Restaurant type selector (7 types)
- Daily covers + staff count inputs
- Labor budget input
- Busy days/times textarea
- Weekly schedule + labor cost % analysis output
