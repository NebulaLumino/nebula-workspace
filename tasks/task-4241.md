## Overview
Build an AI-powered web app that analyzes npm package.json, requirements.txt, go.mod, Cargo.toml, or Gemfile files and identifies available updates with breaking change risk assessment. The AI reads changelogs and release notes to score each update's risk and provides one-click patch generation.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Paste package.json / requirements.txt / go.mod / Cargo.toml / Gemfile / pom.xml
- Fetches latest versions and changelogs from registries
- Breaking change detection: semver analysis + AI-changelog reading
- Risk score per update: Safe, Low Risk, Breaking, Critical
- One-click update with generated codemods/patches
- Transitive dependency vulnerability checking (npm audit style)
- Deprecated package alerts
- Peer dependency conflict detection
- Language/framework: Node.js, Python, Go, Rust, Ruby, Java
- Batch update mode with change summary
- Recommended update order (update order matters for some ecosystems)
- Update all vs selective update UI
