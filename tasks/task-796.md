**Status:** ✅ Complete
**Type:** App Generator — Events/Education
**Priority:** MEDIUM
**Cycle:** 28 (791-820)

## App Concept
Generate workshop curricula and facilitation outlines. Input: workshop topic, audience level (beginner/intermediate/advanced), duration (1hr/half-day/full-day), number of participants, learning objective. Output: complete workshop outline with timed modules, activity descriptions, materials list, breakout exercise prompts, and facilitator notes.

## Tech Stack
- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
- GitHub repo: `ai-workshop-outline`
