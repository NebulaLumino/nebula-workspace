**Status:** ✅ Complete

**Priority:** MEDIUM

## App Concept

Generate user-friendly release notes from changelog entries. Input: version number, release date, list of changes (features, bug fixes, improvements, breaking changes), community contributors. Output: formatted release notes with version header, change categories, user-facing descriptions of each change, upgrade notes, and next-release preview.

- GitHub repo: `ai-release-notes`
