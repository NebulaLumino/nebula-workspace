## Overview
Build an AI-powered web app that converts natural language questions into SQL queries. Users describe what data they want in plain English, optionally provide their schema, and get a syntactically correct SQL query with explanation, sample result structure, and a copy-to-clipboard button.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- CodeMirror or Shiki for syntax highlighting
- OpenAI SDK

## Features
- Natural language input field
- Optional schema paste or CSV upload for context
- Generates SQL (PostgreSQL, MySQL, SQLite, BigQuery dialects)
- Shows query explanation in plain English
- Shows estimated result columns and row sample
- Dialect selector
- Query history saved in localStorage
- Copy query to clipboard
- "Refine query" follow-up input
