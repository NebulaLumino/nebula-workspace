**Status:** ✅ Complete

## Description
Build a Next.js app that generates speech outlines, talking points, analogies, Q&A prep, and visual suggestions for science communication.

## Implementation Notes
- Form inputs: scientific topic, target audience, duration, format (talk/podcast/video/post), tone, presenter background
- Output: timed speech outline with hook/context/conclusion, key talking points, analogies for complex concepts, visual aid suggestions, audience engagement strategies, anticipated Q&A with confidence levels, content adaptations for different formats, accessibility notes
- Uses DeepSeek API via OPENAI_API_KEY env var
- Dark gradient UI with fuchsia/pink theme
