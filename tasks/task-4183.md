## Description
Build an AI-powered CRM data enrichment and deduplication tool that fills in missing contact data, identifies duplicate records, merges them intelligently, and keeps the CRM clean automatically.

## Requirements
- CRM connector (HubSpot, Salesforce, Pipedrive, Airtable)
- AI data enrichment from public sources (LinkedIn, company websites)
- Missing field prediction (title, company size, industry, revenue)
- Duplicate detection using fuzzy matching and AI similarity
- Smart merge with conflict resolution rules
- Data quality scoring per contact
- Stale record detection and re-enrichment triggers
- Enrichment audit log and undo capability
- Bulk enrichment with rate limiting
- Privacy/GDPR compliance (PII masking, consent tracking)

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Clearbit / Apollo API
- Vercel AI SDK
