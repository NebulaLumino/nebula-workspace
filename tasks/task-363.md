## App Concept

A Next.js 16 web app that applies Porter's Five Forces framework to analyze industry competitive dynamics. Users input industry context and get a structured analysis of threat levels across all five forces with strategic implications.

## Features

- Industry selection/input with AI-powered context generation
- Five Forces analysis: Threat of New Entrants, Bargaining Power of Suppliers, Bargaining Power of Buyers, Threat of Substitutes, Competitive Rivalry
- Each force rated (Low/Medium/High/Very High) with detailed explanation
- Industry attractiveness score (1-10)
- Strategic recommendations for each force
- Visual force diagram (SVG-based)
- Competitive strategy suggestions based on force profile

## Tech Stack

- Next.js 16 (App Router)  
- Tailwind CSS  
- DeepSeek API (OpenAI-compatible)  
- TypeScript
