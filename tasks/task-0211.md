## Research Findings — AI SWE Agents 2026

### What Are SWE Agents?
Software Engineering agents that autonomously resolve bugs, implement features, and submit PRs — with minimal human intervention.

### Major Players

| Agent | Creator | What It Does |
|---|---|---|
| **Devin** | Cognition AI | Autonomous SWE, reads codebase, writes code, runs tests, submits PRs |
| **SWE-agent** | princeton-nlp (open source) | Open-source Devin-like, research-grade |
| **SWE-bench** | princeton-nlp | Benchmark: tests agents on real GitHub issues |
| **Aider** | Paul Gauthier | Terminal-based, works with any LLM via API |
| **Claude Code** | Anthropic | CLI agent, can run git/tests/terminal |

### SWE-bench Results (2026)
- Top agents score ~50-60% on real GitHub issues
- Best performers: Claude Agent, GPT-4o, DeepSeek Coder
- Categories: bug fixes, feature additions, refactoring, test writing

### How SWE Agents Work
1. **Read** the codebase and issue description
2. **Plan** the changes needed
3. **Write** code changes
4. **Test** locally (run pytest, node tests, etc.)
5. **Submit** PR or patch

### Limitations
- Struggles with ambiguous requirements
- Can't navigate complex local dev environments easily
- Multi-repo projects are challenging
- Sometimes introduces subtle bugs
- Still needs human review

### Integration with Nova
Nova could build a "SWE Agent Dashboard" — a web UI that:
- Connects to SWE-agent or Aider backend
- Shows active PRs, test status, agent reasoning
- Provides human-in-the-loop review before merge
- Integrates with GitHub org (NebulaLumino)
