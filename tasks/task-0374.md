## App Concept

A Next.js 16 web app that generates deal desk justification memos for sales teams requesting non-standard pricing, discounts, or custom terms. Produces internal documents that help finance and leadership approve deals quickly.

## Features

- Deal details: prospect name, deal value, discount %
- Discount justification: strategic reasons, competitive factors
- Strategic value: logos, references, expansion potential
- Customer financial health indicators
- Competitive context: who's competing, what it takes to win
- Risk assessment: churn risk, payment risk, scope risk
- Approval routing recommendation
- ROI supporting data points

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
