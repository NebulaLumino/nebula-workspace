## Description
Build an AI-powered Personal CFO dashboard and financial advisor. Aggregates bank accounts, investment portfolios, real estate, debt, and expenses into a unified view. AI analyzes spending patterns, generates budget recommendations, optimizes debt payoff strategy (avalanche vs snowball), projects net worth growth, and provides conversational financial advice.

## Requirements
- Bank account aggregation (Plaid integration)
- Investment portfolio tracking (brokerage, 401k, IRA)
- Real estate equity tracking
- Debt payoff optimizer (avalanche, snowball, custom)
- Budget categorization & insights
- Net worth projection (5/10/20/30 years)
- AI conversational advisor (chat interface)
- Monthly financial report generation
- Goal-based savings tracker

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Plaid API
- Alpaca API / Yahoo Finance
- OpenAI SDK (conversational AI)
- Recharts
- SQLite / Prisma for local storage
