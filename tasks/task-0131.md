# Task 131: Explore AI Voice Agents

**Status:** ✅ Completed
**Priority:** LOW
**Type:** Research Task

## Research Summary

### Key Platforms

**Vapi** (vapi.ai)
- Developer-focused API-first voice agent platform
- Acts as orchestration layer connecting STT → LLM → TTS
- Supports OpenAI Realtime API (true speech-to-speech, no pipeline)
- Flexible: plug in OpenAI GPT-4, Claude, Gemini + ElevenLabs/Play.ht voices
- Use case: web apps, phone calls, custom voice assistants

**Bland AI** (bland.ai)
- Phone call automation — inbound + outbound
- Custom voice cloning from short audio samples
- Multi-language support
- CRM/calendar/payment integrations
- SOC2, GDPR, HIPAA compliance options
- Self-hosted infrastructure available

### Real-Time Speech-to-Speech (STS)
- True STS: single multimodal model, no STT→LLM→TTS pipeline
- Target: sub-800ms latency for natural conversation
- OpenAI GPT Realtime API + Gemini 2.5 Flash support this
- ElevenLabs + Resemble AI offer expressive real-time voice synthesis

### Implementation Patterns
- Traditional: Deepgram/STT → GPT-4 → ElevenLabs/TTS
- Modern: OpenAI Realtime API (end-to-end audio)
- Open-source: VALL-E, Tortoise TTS for voice cloning

### Potential Project Ideas
- Voice AI receptionist (Vapi + Twilio)
- AI phone agent for businesses (Bland AI)
- Real-time voice chatbot with custom voice clone
