## App Concept

Generate engaging Instagram captions for posts. Input: photo/video description, post goal (engagement, brand awareness, sales, education), target audience, brand voice, number of hashtags desired, posting time preference. Output: 5 caption variations with different tones (storytelling, witty, instructional, UGC-style, direct), hashtag sets (top 9, 15, 30), and caption structure suggestions (hook, body, CTA, emoji guidelines).

- DeepSeek API (OpenAI-compatible)
