## App Concept

Generate GDPR/CCPA-compliant cookie consent configurations. Input: website type, cookie categories needed (necessary, analytics, marketing, functional, social media), cookie management platform preference (Cookiebot, OneTrust, TrustArc, custom), and banner style preferences. Output: cookie consent banner copy (initial, saved, preferences), cookie categorization descriptions, granular consent options, granular reject-all button, and implementation code snippet guidance.
