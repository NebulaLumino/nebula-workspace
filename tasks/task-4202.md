## Overview
Build an AI-powered web app that recommends the best chart types and dashboard widget layouts for a given dataset. Users paste a dataset preview or describe their data, and the AI recommends visualizations, layout arrangements, and KPI widgets — then renders interactive preview charts.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts or Tremor
- OpenAI SDK

## Features
- Paste CSV header/sample or describe data in natural language
- AI recommends chart types (bar, line, scatter, pie, heatmap, funnel, etc.)
- Recommends layout grid (how many columns, widget sizing)
- Renders live preview charts using real data
- Suggests KPI cards, trend indicators, and filters
- Export dashboard layout as JSON config
- Supports BI tool import (Looker, Tableau style suggestions)
