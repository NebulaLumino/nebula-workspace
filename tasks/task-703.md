## App Concept

Generate testimonial curation strategies and select the right testimonials for each context. Input: list of available testimonials (or customer descriptions), target use context (homepage, pricing page, case study, social proof banner), audience segment. Output: recommended testimonial selection for each context, testimonial display format advice (video vs. text, headshot vs. quote-only), tag categorization by theme (ROI, ease of use, support quality), and a testimonial request prioritization list.
