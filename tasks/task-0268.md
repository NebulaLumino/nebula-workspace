## Research Summary

### The Open Source LLM Landscape (Early 2026)
The open-source LLM ecosystem has exploded, with Meta, Mistral, Alibaba, Google, and others releasing increasingly capable models. The gap between open and closed models has narrowed dramatically, making self-hosted AI viable for most use cases.

### Key Model Families

**Meta Llama 4 (February 2026)**
- Flagship open-source model family
- Sizes: Scout (109B), Maverick (17B), Herd (distributed, 2B+ agents)
- Native multimodal: text, image, video, audio
- 10M token context on Maverick
- Llama 4 Maverick: beats GPT-4o on most benchmarks
- Apache 2.0 license (permissive)
- Native support for tool use and agentic workflows
- Best for: everything from local dev to enterprise deployment

**Mistral Small & Medium**
- Mistral's small-to-medium models: excellent quality/speed/cost ratio
- Mistral 7B: foundational, widely fine-tuned
- Mixtral 8x7B: mixture of experts (MoE), fast inference
- Mistral Small 22B: competitive with GPT-3.5, runs on consumer GPU
- French AI company with European data compliance options

**Alibaba Qwen 3 (2026)**
- Qwen 3.5: 72B model, top-tier reasoning
- Qwen 3: 32B at Q4: excellent for local deployment
- Strong multilingual support (especially Chinese/English)
- Native code generation (matches Claude 3.5 on coding)
- Tool use and agentic capabilities built-in
- Apache 2.0 license
- MoE variants for efficiency

**DeepSeek R1 & V3**
- DeepSeek R1: reasoning model (like OpenAI o1)
- DeepSeek V3: general purpose, 236B params
- Trained at a fraction of the cost of comparable models
- Open weights released
- Strong on math and code
- Already used via API in this very project fleet!

**Google Gemma 3**
- 4B, 12B, 27B variants
- Lightweight but capable
- Good for on-device and edge deployment
- Trained on Google's research

**Nvidia Nemotron**
- 4B and 8B variants optimized for inference
- Strong instruction following
- Used in RTX AI PC features

### Fine-tuning Open Source Models
- **LoRA / QLoRA**: Low-rank adaptation — fine-tune with limited compute
- **Axolotl**: Popular fine-tuning toolkit
- **Unsloth**: Faster fine-tuning on consumer GPUs
- **OpenPipe**: Managed fine-tuning service
- Fine-tune on: domain-specific data, style, personality, tools

### Open Source vs. API Cost Comparison
- GPT-4o: ~$5/1M tokens (input)
- Llama 4 Maverick: free if self-hosted (hardware cost only)
- DeepSeek V3: ~$0.27/1M tokens via API
- Break-even: self-hosting makes sense above ~10M tokens/month

### Relevance to Workspace
The DeepSeek API (already in use) is itself an open-weight model. The AI app fleet could offer a "local mode" using Ollama for users who want privacy or have high volume. Llama 4 and Qwen 3 are now strong enough to replace GPT-4 for most tasks.
