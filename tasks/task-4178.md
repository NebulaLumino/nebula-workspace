## Description
Build an AI-powered competitor pricing intelligence tool that monitors competitor prices across websites, generates pricing change alerts, and provides strategic recommendations for price positioning.

## Requirements
- Competitor website URL monitoring (web scraping)
- Price change detection and alerting (email, Slack, SMS)
- Product matching across competitors (SKU/ASIN mapping)
- Price history visualization and trend charts
- AI-generated competitive pricing analysis report
- Margin impact calculator
- MAP (Minimum Advertised Price) monitoring
- Seasonal pricing pattern detection
- Strategy recommendations (undercut, match, premium positioning)

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Puppeteer / Playwright for scraping
- Supabase for price history
- Vercel AI SDK
