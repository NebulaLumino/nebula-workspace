## App Concept
Generate structured PRDs for product features. Input: feature name, problem it solves, user story, success metrics, technical constraints, dependencies. Output: complete PRD with context and goals, user personas, user stories, feature specifications, acceptance criteria, out-of-scope items, and success metrics.
