## App Concept

Generate comprehensive knowledge base articles from topic descriptions. Input: article title/topic, target audience (end users, developers, admins), complexity level (beginner, intermediate, advanced), article category, key points to cover, tone (formal, friendly, technical). Output: complete knowledge base article with title, summary, step-by-step instructions (with numbered steps), screenshots placeholders, troubleshooting section, related articles links, and metadata (category, tags, difficulty).

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
