# Task 272: AI Upsell/Cross-sell Recommendation Generator

## Type
Directed App (Next.js 16 + DeepSeek API + Vercel)

## Description
Enter a customer's current purchase or subscription → get intelligent upsell and cross-sell recommendations with pricing rationale, positioning copy, and expected conversion lift.

## Specs
- API route: POST /api/generate
- DeepSeek via openai SDK, baseURL: https://api.deepseek.com/v1
- Env: DEEPSEEK_API_KEY
- UI: dark theme with amber/green gradient accents, form → result display

## Priority
MEDIUM
