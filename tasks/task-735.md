## App Concept

Generate optimized social media profile bios. Input: platform (Twitter/X, Instagram, LinkedIn, TikTok), name, profession, key accomplishments, target audience, tone (professional, casual, witty, inspirational), key topics/keywords to include. Output: 5 bio variations with character count compliance, emoji optimization, hashtag suggestions, and link placeholder strategy.
