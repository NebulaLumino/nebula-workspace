## App Concept

Generate a privacy preference center for websites. Input: company name, data categories collected (browsing, purchase, location, demographics), applicable regulations (GDPR, CCPA, COPPA), third-party data sharing partners, cookie categories. Output: full privacy center page with data collection disclosures, consent toggles by category, data deletion request form, cookie preference manager, and GDPR rights explanation (access, rectification, erasure, portability).

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
