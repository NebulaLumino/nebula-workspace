## Project
`ai-persona-builder`

## Description
Build an AI-powered customer segmentation tool that analyzes CRM data, purchase history, and behavioral data to auto-generate customer segments and detailed personas with messaging recommendations.

## Requirements
- CRM data import (HubSpot, Salesforce, CSV)
- Behavioral data import (Google Analytics, Mixpanel)
- AI-powered segment discovery (clustering without manual rules)
- Segment profiling: demographics, behavior, lifecycle stage
- Persona card auto-generation with quotes, pain points, goals
- Channel preference scoring (email, SMS, social, in-app)
- Messaging framework per segment (subject lines, body copy)
- Segment overlap analysis
- Segment health scoring (size, recency, engagement)

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- HubSpot / Salesforce API
- Vercel AI SDK
