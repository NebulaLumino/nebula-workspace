## Overview
Build an AI-powered web app that helps developers find the right open source project to contribute to and understand what their contributions should look like. Users paste a GitHub URL or describe their skills, and the AI analyzes the repo's issue tracker, PR history, CONTRIBUTING.md, code of conduct, and existing contribution patterns to suggest specific issues to tackle and how to approach them.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- GitHub repo URL → full contribution analysis
- Good first issue finder: analyzes closed PRs to find similar patterns
- PR template generator: suggest what to include in a PR for this repo
- Code style analyzer: matches the project's style from existing code
- Reviewer suggestion: who typically reviews this type of change
- Contribution gap analysis: docs, tests, types that are missing
- Community health assessment: response time, review speed, welcoming signals
- Language/tech stack alignment with user-described skills
- Fork + branch strategy guidance
- CLA/DCO requirement detection and guidance
- "What not to do" warnings based on closed rejected PRs
