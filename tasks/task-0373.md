## App Concept

A Next.js 16 web app that generates compelling executive summaries for client proposals and RFPs. Users input the client context, proposed solution, and scope; the AI outputs a tight, persuasive summary designed to get the "yes."

## Features

- Client info: name, industry, decision-maker
- Problem/situation description
- Proposed solution overview
- Scope of work summary
- Key benefits and ROI framing
- Investment/pricing section
- Timeline overview
- Why us / differentiator paragraph
- Call to action / next steps

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
