# Task 799: AI Performance Improvement Plan Generator

**Status:** ✅ Complete
**Type:** App Generator — HR/Management
**Priority:** MEDIUM
**Cycle:** 28 (791-820)

## App Concept
Generate structured Performance Improvement Plans (PIPs). Input: employee name, role, performance issues (specific), manager name, PIP duration (30/60/90 days), improvement goals. Output: complete PIP document with performance issues documented, measurable improvement goals, weekly milestones, resources/support provided, and consequences of non-completion.

## Tech Stack
- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
- GitHub repo: `ai-performance-improvement-plan`
