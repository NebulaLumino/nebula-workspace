## App Concept

A Next.js 16 web app that generates 3-paragraph brand positioning statements using the proven formula: "For [target audience], [brand] is the [category] that [key benefit]. Unlike [competitor], we [key differentiator]." Users input brand details and get multiple polished positioning variants.

## Features

- Multi-field input: brand name, target audience, category, key benefit, emotional hook, competitor, differentiator
- 3 variants: formal, punchy, narrative
- Tagline generation (5 options)
- Unique Value Proposition (UVP) statement
- Messaging hierarchy: headline, subhead, body copy
- One-sentence positioning summary
- Copy-to-clipboard for each section

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
