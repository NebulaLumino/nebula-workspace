## Overview
Build an AI-powered web app that takes raw security scan output (from Snyk, Dependabot, npm audit, Trivy, Grype, Bandit, or Semgrep) and generates human-readable vulnerability summaries with prioritized remediation steps. Users paste scan results or JSON output, and the AI prioritizes by CVSS score, explains each vulnerability in plain language, and generates specific fix commands or code patches.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Paste raw scan output (JSON, SARIF, or plain text) from: Snyk, Dependabot, npm audit, yarn audit, pip audit, Trivy, Grype, Bandit, Semgrep, OWASP ZAP, nmap
- CVSS score explanation in plain language
- Prioritized fix queue: critical → high → medium → low
- Specific remediation: upgrade to version X, apply patch, use alternative package
- Attack scenario explanation: how can this vulnerability be exploited
- Exploitability assessment (is there a known exploit in the wild?)
- False positive detection and suppression guidance
- Security advisory references (CVE, GHSA, OSV)
- Regulatory compliance mapping: PCI-DSS, SOC2, HIPAA, GDPR implications
- Batch fix command generation (pip install -r requirements.txt --upgrade, npm audit fix)
- Fix effort estimation: quick (1-line change) vs major (architectural change)
- Summary dashboard: vulnerability count by severity, category, and affected file
- Historical trend: are vulnerabilities increasing or decreasing across scans
