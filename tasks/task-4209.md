## Overview
Build an AI-powered cohort retention analyzer. Users input user signup dates and activity dates, the app builds a retention heatmap table, and the AI interprets retention patterns, identifies anomalies, and suggests what behaviors drive long-term retention.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (for heatmap visualization)
- OpenAI SDK

## Features
- Input: user_id, cohort_period (signup date), activity_period
- Generates retention heatmap table (cohorts × periods)
- Color-coded retention percentages
- Flattening point identification (when churn stabilizes)
- Best/worst performing cohorts highlighted
- AI interprets: "Cohorts from March show 15% lower retention — possible cause: [X]"
- Weekly, monthly, and daily period options
- Export heatmap as CSV or image
