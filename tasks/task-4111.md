## Description
An AI-powered tool that guides users through Cognitive Behavioral Therapy (CBT) thought reframing exercises. Users input distressing thoughts, and the AI helps identify cognitive distortions, challenge negative beliefs, and generate balanced, rational alternatives.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User enters a distressing automatic thought
2. AI identifies the cognitive distortion type (catastrophizing, black-and-white thinking, etc.)
3. AI guides user through Socratic questioning to challenge the thought
4. AI generates a balanced, reframed thought
5. User rates distress before/after to measure progress
