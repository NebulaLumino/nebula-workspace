## Description
Build an AI-powered decentralized identity (DID) verification helper tool. Helps users create, manage, and verify DIDs using protocols like ENS, Lens Protocol, Web5, and Veramo. Features include DID document generation, credential verification, identity linkage analysis, and compliance-ready identity reports for DeFi compliance (travel rule, KYC).

## Requirements
- DID creation (ETH DID, did:key, did:web)
- ENS domain resolution & linking
- Lens Protocol profile integration
- Verifiable Credential (VC) verification
- DID document parsing & validation
- Identity linkage graph visualization
- Travel rule compliance helper
- Credential QR code generation
- Identity reputation scoring

## Tech Stack
