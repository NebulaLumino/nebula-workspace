## Overview
Build an AI-powered web app that generates CI/CD pipeline configurations from a description of the project and deployment needs. Users describe their stack (e.g., "a Next.js app deploying to Vercel with Playwright e2e tests, Lighthouse CI, and Slack notifications on failure"), and the AI generates complete GitHub Actions, GitLab CI, or Jenkins pipelines with testing, building, and deployment stages.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Natural language → CI/CD pipeline config
- Platform: GitHub Actions, GitLab CI, Jenkinsfile, CircleCI, Bitbucket Pipelines
- Language detection: Node.js, Python, Go, Rust, Docker, static sites
- Stages: lint, test (unit, integration, e2e), build, security scan, deploy
- Matrix strategy for multi-version/multi-platform testing
- Caching strategies: npm cache, Docker layer cache, pip cache
- Environment secrets management guidance
- Conditional deployments (branch-based, tag-based, manual approval)
- Notification integrations: Slack, Discord, email, Microsoft Teams
- Lighthouse CI integration for performance budgets
- Security scanning: Trivy, Snyk, npm audit, Bandit
- Docker build and push with layer caching
- Terraform/ Pulumi deployment stage generation
- Vercel, Netlify, AWS, GCP deployment configs
