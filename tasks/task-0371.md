## App Concept

A Next.js 16 web app that generates comprehensive sales playbooks including ideal customer profiles, discovery questions, objection handling scripts, and closing techniques. Built for B2B SaaS and enterprise sales teams.

## Features

- ICP definition: company size, industry, tech stack, pain points
- Discovery questions: 10-15 open-ended, probing questions
- Objection handling: top 5 objections + response scripts
- Close techniques: 3-4 specific close methods with when to use
- Sales stages: step-by-step pipeline from lead to close
- Competitive battle cards: how to position vs key competitors
- Talk tracks and email sequences

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
