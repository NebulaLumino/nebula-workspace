# Task 1215: AI Retail Inventory Management & Reorder Point Generator

**Status:** ✅ Complete

## Summary
Built `ai-retail-inventory` — Next.js AI app that generates inventory management plans, reorder point calculations, and SKU classification reports.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript + Tailwind CSS
- DeepSeek API (deepseek-chat model)
- Dark gradient UI (violet accent)

## GitHub
https://github.com/NebulaLumino/ai-retail-inventory

## Features
- Retail business type selector (8 types)
- SKU count + supplier lead time inputs
- Stock issues textarea
- Annual sales volume input
- ROP/EOQ calculations + ABC analysis output
