# Task 271: AI Customer Journey Map Generator

## Type
Directed App (Next.js 16 + DeepSeek API + Vercel)

## Description
Build a web app where users input their product and customer stages → the AI generates a complete visual customer journey map with touchpoints, emotions, pain points, and opportunities at each stage.

## Prompt for Build
- Product name, customer type, key stages (awareness, consideration, purchase, retention, advocacy)
- Output: formatted markdown journey map with stage descriptions, customer actions, emotions, touchpoints, pain points, opportunities, and recommended actions
- UI: dark themed, gradient bg, journey stages in visual cards or flow layout, copy button

## Specs
- API route: POST /api/generate
- DeepSeek model: deepseek-chat via openai SDK
- Env var: DEEPSEEK_API_KEY
- Vercel env: set DEEPSEEK_API_KEY in project settings
