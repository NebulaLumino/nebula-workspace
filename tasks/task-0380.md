## App Concept

A Next.js 16 web app that generates structured win/loss analysis reports capturing deal outcomes, competitive factors, and actionable lessons. Used by sales leaders to learn from every deal and improve win rates.

## Features

- Deal snapshot: company, competitor, deal size, stage lost/won
- Outcome summary: won/lost, margin, timeline
- Competitive factors: who competed, pricing, features, relationship
- Sales process evaluation: discovery, demo, proposal, negotiation
- What went right / what went wrong
- Competitor strengths and weaknesses observed
- Lessons learned (3-5 actionable takeaways)
- Recommended process improvements

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
