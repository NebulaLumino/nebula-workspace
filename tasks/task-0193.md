## Description

- Dify, Coze, LangFlow platforms

---

## Research Findings — No-Code/Low-Code AI Platforms 2026

### Visual Workflow Builders

**Zapier + AI**
- "AI Actions" in Zapier: LLM step inside any Zap
- Connect AI to 6,000+ apps
- Best for: Automating workflows with AI (lead enrichment, content generation)
- Limitations: Linear, not good for complex state

**Make (formerly Integromat)**
- More visual than Zapier, supports loops/conditionals
- AI modules for text, image generation
- Best for: Complex multi-step automations

**n8n**
- Open source, self-hostable workflow automation
- AI nodes for OpenAI, Anthropic, local LLMs
- Best for: Technical teams wanting full control

### AI Agent Builders (No-Code)

**Coze (ByteDance)**
- Build AI chatbots with memory, knowledge bases, plugins
- Publish to Discord, Slack, Telegram, websites
- Best for: Non-technical teams building customer support bots
- Free tier available

**Dify**
- Open source LLM app development platform
- Visual workflow editor, RAG pipeline, prompt engineering
- Deploys to your own infrastructure
- Best for: Teams wanting self-hosted AI apps

**LangFlow**
- Visual version of LangGraph
- Drag-and-drop AI workflow designer
- Best for: Developers prototyping LangChain pipelines

**Gumloop**
- Visual AI agent builder
- Chain multiple agents together visually
- Best for: Non-engineers building agentic workflows

### When to Use No-Code vs. Custom

| Situation | Use No-Code | Use Custom Code |
|---|---|---|
| Simple chatbot | Coze, Zapier | LangChain |
| RAG over documents | Dify | LangChain + Pinecone |
| Complex multi-agent | Gumloop | AutoGen/CrewAI |
| Full-stack web app | Bubble + AI | Next.js + Vercel |
| Internal tool | Dify, n8n | FastAPI + React |

### For Nova
Dify or Coze could be interesting for rapidly prototyping AI workflows without full custom code.
