## Overview
Build an AI-powered web app that automatically generates a comprehensive data dictionary from any dataset. Users upload a CSV or paste column names and sample values, and the AI generates descriptions, data types, valid value ranges, example values, and business context for every column.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK

## Features
- CSV upload or paste column headers + sample rows
- AI generates: column description, data type, format, sample values
- Flags: primary keys, foreign keys, date columns, numeric vs categorical
- Valid value enumeration for categorical columns
- Null/missing frequency analysis
- Export as Markdown table, JSON schema, or CSV
- Business context suggestions for each column
- Editable AI descriptions (user can refine)
- Links to similar common data warehouse field conventions
