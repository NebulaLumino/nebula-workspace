`ai-meeting-notes`

## Description
Build an AI-powered meeting notes tool that records, transcribes, and generates structured summaries with extracted action items, owners, deadlines, and decisions. Integrates with Zoom, Google Meet, Teams, and provides follow-up email drafting.

## Requirements
- Video/audio meeting integration (Zoom, Google Meet, Teams)
- Real-time transcription (Whisper API)
- AI summarization with topic segmentation
- Action item extraction with owner assignment
- Deadline detection and calendar integration
- Decision logging and decision-maker tracking
- Follow-up email/comms draft generator
- Searchable meeting archive

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Whisper API for transcription
- Google Calendar / Outlook API
- Vercel AI SDK
