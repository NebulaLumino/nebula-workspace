# Task 241: AI Agent Long-Term Memory

**Status:** [ ]
**Priority:** HIGH
**Type:** Implementation Project

## Description
Build a persistent memory layer for AI agents — stores conversations, learns user preferences, and retrieves relevant memories across sessions.

## Requirements
- Store conversation summaries and key facts
- Add/edit/delete memory entries with categories
- Semantic search across memories (query → relevant memories)
- "Learn from conversation" button to extract key facts
- Memory usage statistics
- DeepSeek for embedding + retrieval
