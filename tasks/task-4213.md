## Overview
Build an AI-powered survey analysis tool. Users upload survey responses (CSV with open-ended text answers), and the app clusters responses by theme, performs sentiment analysis per question, and generates a visual summary dashboard with AI-written key findings.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- OpenAI SDK

## Features
- CSV upload: multiple open-ended text columns
- AI sentiment score per response (positive/neutral/negative) with % breakdown
- Theme clustering: groups similar responses into topics
- Bar chart of sentiment per question
- Word cloud per cluster (top frequent terms)
- AI-generated summary for each cluster: "Theme: Pricing concerns — 40% of respondents mentioned cost"
- Demographic breakdown: filter by age/gender/region if columns exist
- Export: full report as Markdown, sentiment scores as CSV
