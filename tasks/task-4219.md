## Overview
Build an AI-powered data quality scoring tool. Users upload a dataset, and the app evaluates it across multiple data quality dimensions (completeness, accuracy, consistency, timeliness, uniqueness, validity) and generates a visual data quality scorecard with per-column grades and AI-written improvement recommendations.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts (radar chart for scorecard)
- OpenAI SDK

## Features
- CSV upload (or paste schema)
- Evaluates: completeness (% null/empty), uniqueness (duplicate %), validity (format errors), timeliness (date freshness), consistency (across related columns)
- Overall DQ score (0-100) with letter grade (A-F)
- Per-column breakdown: which columns are clean vs dirty
- AI improvement recommendations per column
- Benchmark comparison: how does this dataset compare to typical industry standards
- Score trend over time (if historical data is available)
- Export: scorecard as PDF-ready HTML, recommendations as CSV
