# Task 1212: AI Food Recipe Scaling & Restaurant Menu Cost Generator

**Status:** ✅ Complete

## Summary
Built `ai-recipe-scaler` — Next.js AI app that scales recipes and calculates food costs/menu pricing for restaurants.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript + Tailwind CSS
- DeepSeek API (deepseek-chat model)
- Dark gradient UI (green accent)

## GitHub
https://github.com/NebulaLumino/ai-recipe-scaler

## Features
- Cuisine type selector (10 cuisines)
- Original/target servings inputs
- Ingredient list textarea
- Food cost margin selector
- Scaled recipe + cost breakdown output
