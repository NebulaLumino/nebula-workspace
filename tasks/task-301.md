## Type
Directed App (Next.js 16 + DeepSeek API + Vercel)

## Description
Enter company name, product, target audience, and onboarding goal → generate a complete email drip sequence (welcome, feature highlights, tips, case study, upgrade CTA). Includes send-time recommendations.

## Implementation Notes
- Next.js 16 App Router
- DeepSeek API for generation
- Vercel deployment
- Multiple sequence templates (trial → conversion, free → paid, enterprise onboarding)
- Copy-to-clipboard for each email
- Export full sequence as markdown
