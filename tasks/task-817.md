# Task 817: AI Net Promoter Score Follow-up Generator

**Status:** ✅ Complete
**Type:** App Generator — Customer Success
**Priority:** MEDIUM
**Cycle:** 28 (791-820)

## App Concept
Generate NPS follow-up sequences based on score. Input: NPS score (0-10), verbatim comment, customer name, time since last interaction. Output: automated follow-up sequence — promoters (ask for referral/testimonial), passives (offer value to elevate), detractors (apology + recovery plan). Includes response templates at 3 urgency levels.

## Tech Stack
- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
- GitHub repo: `ai-nps-followup`
