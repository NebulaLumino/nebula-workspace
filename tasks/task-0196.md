## Research Findings — AI Coding Assistants 2026

### The Landscape

| Tool | Type | Key Strength | Price |
|---|---|---|---|
| **Cursor** | IDE (VS Code fork) | Agent mode, team policies, best-in-class | $20/mo |
| **GitHub Copilot** | IDE plugin | Ubiquitous, decent quality | $10/mo |
| **Claude Code** | CLI agent | Deep research, autonomous coding | $100/mo (Pro) |
| **Windsurf (Cascade)** | IDE plugin | Agentic, good UI | Free + $15/mo |
| **Continue** | IDE plugin | Open source, self-hosted | Free |
| **Codeium** | IDE plugin | Generous free tier | Free |

### Cursor (Best Overall 2026)
- Agent mode: autonomous plan → implement → test
- Team Policies: enforce code standards for entire team
- Apply: review + edit across entire codebase
- Built on VS Code — familiar, powerful

### Claude Code
- CLI agent for autonomous development
- Can run git commands, tests, terminal operations
- Best for: complex tasks requiring multi-file changes
- No UI — terminal only

### Open Source Alternatives
- **Continue.dev**: Open-source Copilot alternative, works with any LLM including local Ollama
- **Tabnine**: Long-standing, now includes agentic features
- Best for: teams wanting self-hosted / privacy

### Specialized Coding Agents
- **Devin** (Cognition): Autonomous SWE agent, live on GitHub
- **Aider**: Terminal-based, works with any LLM via API
- **SWE-agent**: Open-source Devin-like for research
