## Description
An AI-powered role-play simulator for practicing social skills. Users select a social scenario (job interview, first date, difficult conversation, networking event, etc.), and the AI plays the other party, providing realistic responses and feedback on the user's communication effectiveness.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User selects a social scenario and difficulty level
2. AI plays the conversation partner with appropriate personality
3. User responds to AI prompts in a chat-like interface
4. After the conversation, AI provides detailed feedback
5. User can replay with different approaches
