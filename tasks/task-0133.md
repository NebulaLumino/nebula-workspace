## Research Summary

### What Are Multimodal Embeddings?
Unified vector representations that capture text, images, audio, and video in the same semantic space. Enables cross-modal search ("find the image that matches this text").

### Key Models

**CLIP** (OpenAI)
- Maps images + text to same embedding space
- Open source (ViT-B/32, ViT-L/14 variants)
- Foundation for most multimodal retrieval systems
- Use: image search, zero-shot image classification

** ALIGN ** (Google)
- Large-scale image-text matching
- 1B+ training pairs
- Powers Google Search visual matching

**Sentence CLIP**
- Optimized for sentence-level semantic matching
- Better for text-to-image retrieval than vanilla CLIP

### Tools & Libraries
- **transformers** (HuggingFace): CLIP models readily available
- **FAISS / Milvus**: vector storage for embedding similarity search
- **Clip-as-service**: deploy CLIP embeddings at scale

### Project Ideas
- Image search engine (upload photo, find similar)
- Multimodal RAG: retrieve relevant images + text for AI Q&A
- Cross-modal content moderation
