## Description
Build a Next.js app that generates structured peer review feedback with strengths, weaknesses, questions, recommendations, and scores.

## Implementation Notes
- Form inputs: manuscript text, manuscript title, reviewer expertise area, review criteria type
- Output: manuscript summary, overall recommendation, criteria scores (7 dimensions), major strengths, weaknesses, specific comments, questions for authors, required/suggested revisions
- Uses DeepSeek API via OPENAI_API_KEY env var
- Dark gradient UI with yellow/amber theme
