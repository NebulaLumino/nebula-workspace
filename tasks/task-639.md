## App Concept

Analyze existing landing pages and generate improved versions. Input: URL of existing landing page, target audience description, primary conversion goal, improvements needed (headline, CTA, layout, copy). Output: detailed analysis of current page strengths/weaknesses, improved headline options, CTA variations, section-by-section copy improvements, and recommended layout changes.

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
