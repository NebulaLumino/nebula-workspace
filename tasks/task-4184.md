## Description
Build an AI-powered vendor management tool that tracks vendor performance across delivery, quality, cost, and responsiveness, generates scorecards, and provides recommendations for renewals and alternatives.

## Requirements
- Vendor profile management and contract tracking
- Performance dimension scoring: Quality, Delivery, Cost, Communication, Compliance
- AI-generated performance narratives from raw data
- SLA breach detection and alerting
- Comparative vendor ranking
- AI recommendation engine (renew, renegotiate, replace)
- Historical performance trend charts
- Vendor risk assessment (financial health, dependency)
- RFx document generation (RFP, RFQ)
- Contract renewal calendar

- OpenAI SDK (GPT-4o)
