## Description
An AI-powered mental health check-in tool for new parents navigating the postpartum period. Daily check-ins screen for postpartum depression and anxiety, generate personalized self-care recommendations, and connect users with professional resources, support groups, and helplines.

## Workflow
1. User completes daily mental health check-in
2. AI screens for PPD/PPA symptoms and tracks over time
3. AI generates personalized self-care recommendations for new parents
4. Resource finder matches user with local/online support
5. Escalation to crisis resources if needed
