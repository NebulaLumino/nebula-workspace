## Concept & Vision
A tool for museum directors, cultural institution boards, and museum consultants to develop comprehensive strategic plans. The output follows American Alliance of Museums (AAM) strategic planning standards — covering mission, vision, strategic priorities, DEAI goals, and financial sustainability.

## Specific Features
- Strategic framework generator (mission, vision, values)
- SWOT analysis builder
- Strategic priority generator (3-5 major goals)
- DEAI commitment statement builder
- Audience development strategy
- Revenue diversification plan (earned income, contributed, endowment)
- Collection development policy brief
- Board governance enhancement suggestions

## Input Fields
- Institution type (art museum / history museum / science center / zoo / botanical garden / historic house)
- Institution size (annual budget range)
- Current mission statement (or description)
- Geographic context (urban / suburban / rural)
- Key challenges (attendance / funding / relevance / DEAI / facilities)
- Strategic planning timeframe (3-year / 5-year / 10-year)
- Board composition notes

## Output
Strategic plan document:
- Executive summary
- Organizational overview
- SWOT analysis
- Vision and values
- Strategic priorities (3-5 goals) with objectives and tactics
- DEAI goals and commitments
- Audience development strategy
- Financial sustainability plan
- Implementation timeline
- Board governance enhancements

## AI Model
DeepSeek via `/api/generate`.

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI.
