## Concept & Vision
A specialized tool for aerospace engineers, aviation researchers, and defense contractors to generate technical briefs, system requirements documents, and design analysis reports. Feels like precision engineering — every statement is technically precise, every claim is substantiated with data or first principles.

## Specific Features
- System requirements document (SysRD) generator
- Design analysis report (DAR) generator
- Flight envelope characterization brief
- Structural load analysis summary
- Propulsion system specification generator
- Avionics requirements brief
- Safety and redundancy analysis (FMEA/FMECA framework)

## Input Fields
- System/platform type (fixed-wing / rotorcraft / UAV / spacecraft / missile)
- Design phase (conceptual / preliminary / detailed)
- Mission profile description
- Operating environment (altitude, temperature range, humidity, G-forces)
- Key performance parameters (speed, range, payload, endurance)
- Regulatory framework (FAR / MIL-SPEC / EASA / NASA standards)

## Output
Technical engineering document:
- System overview and requirements
- Design justification
- Performance analysis summary
- Safety and redundancy analysis
- Compliance matrix
- Recommended next steps

## AI Model
DeepSeek via `/api/generate`.

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI.
