## Concept & Vision
A tool for urban planners, city governments, and developers to generate zoning analysis reports and urban planning briefs. The output follows standard urban planning document formats — covering land use, density, transportation, infrastructure capacity, and community impact. Feels like a professional city planning department document.

## Specific Features
- Zoning district analysis generator
- Land use compatibility assessment
- Density and FAR (floor area ratio) calculator
- Transportation impact summary
- Infrastructure capacity analysis (water, sewer, power)
- Affordable housing requirement calculator
- Community facilities needs assessment
- General plan consistency analysis

## Input Fields
- Project type (residential / commercial / mixed-use / industrial / public)
- Parcel address and current zoning
- Proposed development size (sq ft, number of units, height)
- Site area (acres)
- City/county jurisdiction
- Applicable zoning code section
- Surrounding land uses
- Transportation context (transit proximity, road classification)

## Output
Zoning analysis report:
- Site and context overview
- Current zoning compliance analysis
- Density and dimensional analysis
- Use compliance review
- Required entitlements/approvals
- Infrastructure capacity assessment
- Recommendations and next steps

## AI Model
DeepSeek via `/api/generate`.

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI.
