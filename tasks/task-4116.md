## Description
An AI-powered tool that generates personalized guided meditation and mindfulness sessions on-demand. Users specify their current emotional state, time available, and meditation goal, and the AI creates a custom script with breathing patterns, visualization guidance, and body scan instructions.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User selects current emotional state, time available, and meditation type
2. AI generates a personalized guided meditation script
3. UI presents the session with a timer and step-by-step guidance
4. User can adjust session length, voice, and focus area
5. Session history and favorites are saved
