## App Concept

A Next.js 16 web app that generates executive summaries from raw business data (pitch decks, meeting notes, financial data). Users input company information and the AI produces a polished, professional executive summary suitable for investors, board members, or stakeholders.

## Features

- Multi-section input form (company overview, problem, solution, traction, financials)
- AI-powered executive summary generation with configurable tone (formal, persuasive, technical)
- Export to PDF and copy-to-clipboard
- Templates for different use cases (investor pitch, board update, internal strategy)
- Word count and reading time estimate

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
