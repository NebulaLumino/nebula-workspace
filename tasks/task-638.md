## App Concept

Generate robots.txt files and robots meta tag recommendations. Input: website type, URLs to allow/block (admin, staging, search, feeds, APIs), sitemap URL, crawler-specific rules (Googlebot, Bingbot), pagination handling preferences. Output: complete robots.txt file content, page-by-page meta tag recommendations (index/noindex, follow/nofollow), canonical URL guidance, and hreflang tag recommendations for international sites.
