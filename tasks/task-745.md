## App Concept

Generate executive summaries from detailed reports or proposals. Input: document type (proposal, report, strategy doc), target audience (C-suite, board, investor), key sections to include, tone (formal, concise, persuasive), length limit. Output: compelling executive summary with key finding highlighted, bottom-line recommendation, supporting evidence summary, and visual placeholder suggestions. Multiple length variations (30-second read, 2-minute read).
