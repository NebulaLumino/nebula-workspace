## Overview
Build an AI-powered web app (and CLI tool) that generates conventional git commit messages and GitHub PR descriptions from staged changes. Users can paste diffs, link a GitHub PR URL, or integrate via CLI pre-commit hook. The AI understands conventional commits (feat, fix, chore, docs, refactor, test, perf, breaking change) and generates human-readable yet concise messages.

## Tech Stack
- Next.js 15 (App Router) + Node.js CLI
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Paste diff / unified patch → AI commit message
- Conventional commit format (feat:, fix:, docs:, refactor:, test:, chore:)
- Multi-commit batch mode: multiple diffs → multi-commit list
- PR description generator: bullet points of changes, breaking changes, testing notes, screenshots
- Scope inference from changed files/modules
- Breaking change detection and flagging
- CLI integration: `npx ai-commit` as pre-commit hook
- Commit history analyzer: suggest commit message style based on project history
- Copilot-style suggestion as user types a commit message
