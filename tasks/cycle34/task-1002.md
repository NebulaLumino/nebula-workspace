# Task 1002: AI Court Ruling Research & Case Similarity Finder

**Status:** ✅ Complete
**Priority:** HIGH
**Cycle:** 34 (1001-1030)
**Theme:** Government, Law & Civic Technology

---

## Concept & Vision
A tool for attorneys, law students, and self-represented litigants to research case law. Describe a legal issue and get relevant precedents, holdings, and reasoning patterns. The output is a structured legal memo with case citations and applicability analysis.

## Specific Features
- Issue summarizer from fact pattern description
- Relevant precedent finder (based on legal issue)
- Case holding synthesizer
- Analogical reasoning builder (how cases apply to facts)
- Dissent and concurrence analysis
- Circuit/split identifier (are courts disagreeing?)
- Citation chain builder (what cases cite this case?)
- Judge tendency analyzer (optional)

## Input Fields
- Jurisdiction (federal circuit / state / appellate / trial)
- Legal issue area (constitutional / tort / contract / criminal / administrative)
- Fact pattern description
- Specific question presented
- Preferred courts (e.g., 9th Circuit, NY Court of Appeals)

## Output
Legal research memo:
- Issue statement
- Rule statements (black letter law)
- Case analysis (3-5 key cases with facts, holding, reasoning)
- Application (how cases apply to the presented facts)
- Strategy recommendations
- Counterarguments and risks
- Key citations

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, dark gradient UI, DeepSeek API.
