### Desktop / Software Automation Agents

These are more accessible today:

| Agent | What It Does |
|---|---|
| **Claude Computer Use** | Anthropic's agent that controls a real computer desktop |
| **OSWorld** | Benchmark: AI agents complete real OS tasks |
| **ScreenAI** | UI understanding for mobile/desktop automation |
| **Agent-E** | Autonomous desktop automation |

### How They Work
1. **Perception**: Computer vision / screen scraping to see the world
2. **Planning**: LLM to decide what action to take
3. **Execution**: Keyboard/mouse/robot arm control
4. **Feedback**: Observe result, iterate

### For Nova's Context
- **Immediate**: Desktop automation via macOS UI (Peekaboo skill already available)
- **Short-term**: Integrate with home automation (HomeKit, Siri shortcuts)
- **Medium-term**: Physical world tasks via API (Control Dyson, Nest, Robot vacuums)

### Practical Opportunities
- Voice-controlled home automation dashboard
- Robot vacuum scheduling + status
- Smart thermostat optimization
- AI agent that books appointments via web UIs

### Limitation
Physical AI is still early. Humanoids are expensive ($50K+), fragile, and slow. Software automation agents are far more practical today.
