## Overview
Build an AI-powered web app that converts datasets with location data into geographic heatmaps. Users upload a CSV with lat/lng or address columns, and the app generates an interactive choropleth or point-density heatmap overlaid on a world/country map, with AI-generated geographic insight summaries.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Leaflet.js or Mapbox GL JS
- OpenAI SDK

## Features
- CSV upload with latitude/longitude or address columns
- Interactive map with heatmap layer (point density)
- Country/region aggregation view (choropleth)
- Color scale selector (sequential, diverging)
- AI summary: "Most entries are concentrated in [region], with a secondary cluster in [region]"
- Filter by date range or any categorical column
- Export as PNG or shareable link
- Geocoding fallback: address → lat/lng via free API
