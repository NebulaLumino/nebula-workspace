# Task 376: AI Quarterly Business Review (QBR) Generator

**Status:** ⏳ Pending  
**Type:** App Generator — Business Operations  
**Priority:** MEDIUM  

## Context
A structured QBR deck generator for presenting quarterly results to leadership or stakeholders.

## Spec
- **Input:** Company/team name, quarter, key metrics (revenue, growth, churn), wins, challenges, next quarter goals
- **Output:** Polished QBR presentation with executive summary, metrics dashboard, analysis, and forward-looking plan
- **Sections:** Executive summary, metric highlights, goal progress, root cause analysis, lessons learned, next quarter plan

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK
