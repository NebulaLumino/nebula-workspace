## App Concept

Generate comprehensive data dictionaries for databases or data models. Input: table names, column names, data types, brief descriptions, related tables. Output: polished data dictionary with field descriptions, data types, sample values, relationships, business definitions, and quality notes. Exportable as markdown table or PDF.
