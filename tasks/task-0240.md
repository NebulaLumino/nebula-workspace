## Research Findings — AI Ecosystem & Tooling 2026

### LLM API Providers

| Provider | Model | Best For | Price |
|---|---|---|---|
| **OpenAI** | GPT-4o, o3 | General, reasoning | $5-15/1M tokens |
| **Anthropic** | Claude 4 Sonnet | Long docs, coding | $3-15/1M tokens |
| **Google** | Gemini 2.0 Flash | Multimodal, speed | $0.075/1M tokens |
| **DeepSeek** | V3 | General, coding | $0.27/1M tokens |
| **Cohere** | Command R+ | RAG, enterprise | $1-3/1M tokens |
| **Mistral** | Mistral Large | European, multilingual | $2-8/1M tokens |
| **Together AI** | Mixtral, Qwen | Open models, cheap | $0.50/1M tokens |

### Vector Databases

| DB | Type | Open Source | Best For |
|---|---|---|---|
| **Pinecone** | Managed | No | Production at scale |
| **Weaviate** | Open source | Yes | Hybrid search, GraphQL |
| **Qdrant** | Open source | Yes | High-performance, self-hosted |
| **Chroma** | Open source | Yes | Local prototyping |
| **Milvus** | Open source | Yes | Massive scale |
| **pgvector** | Postgres ext | Yes | Small-medium, existing PG infra |

### For Nova's Infrastructure

**Recommended stack:**
- LLM: DeepSeek (current) + Anthropic Claude (high-quality tasks)
- Orchestration: LlamaIndex for RAG (already has data)
- Observability: Helicone (simple, effective)
- Vector DB: Qdrant (self-hosted, fast)
- Evals: Braintrust for A/B testing model outputs
