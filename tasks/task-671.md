## App Concept

Generate professional case study templates for customer success stories. Input: customer company name, industry, challenge faced, solution implemented, metrics improved (with before/after numbers), testimonial quote, key learnings. Output: polished case study template with headline, executive summary, challenge/solution/results sections, pull-quote formatting, data visualization placeholders, and SEO-optimized subheadings. Multiple layout options (short/medium/long form).

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
