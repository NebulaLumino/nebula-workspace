## Description
Next.js app that generates comprehensive technical blog posts with AI. Input: topic, target audience, tone, length, key points. Output: full blog post with headings, intro, body, conclusion, call-to-action.

## Implementation Notes
- Dark gradient UI with blue/purple gradient theme
- Form: topic, audience (developers/managers/students/execs), tone (technical/conversational/educational), length (short/medium/long), key points textarea
- DeepSeek API via OPENAI_API_KEY env var
- Lazy-initialized OpenAI client (avoids build-time errors)
