## App Concept

A Next.js 16 web app that generates compelling brand positioning statements using the brand positioning formula. Users input their brand, target audience, category, key benefit, and emotional hook; the AI produces multiple positioning statement variants.

## Features

- Multi-field input: brand, target audience, category, main benefit, emotional hook, differentiator
- 3 variants of positioning statement (formal, punchy, narrative)
- Tagline generation (5 options)
- UVP (Unique Value Proposition) statement
- Messaging hierarchy: headline, subhead, body copy
- One-sentence positioning summary for internal alignment
- Copy-to-clipboard for each section
- A/B test variants

## Tech Stack

- Next.js 16 (App Router)  
- Tailwind CSS  
- DeepSeek API (OpenAI-compatible)  
- TypeScript
