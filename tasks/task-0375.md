## App Concept

A Next.js 16 web app that generates structured 30-60-90 day customer success plans covering onboarding milestones, adoption goals, and expansion opportunities. Used by CS teams to ensure new customers get value quickly and stick long-term.

## Features

- Customer snapshot: company, industry, use case, success criteria
- 30-day plan: onboarding tasks, setup milestones, training schedule
- 60-day plan: adoption targets, integration checklist, health metrics
- 90-day plan: expansion triggers, renewal check-in, advocacy goals
- QBR schedule recommendations
- Risk indicators to watch
- Escalation triggers
- Success metric definition

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
