## App Concept

A Next.js 16 web app that converts complex partnership agreement terms into plain-English summaries. Users paste or describe the agreement terms; the AI outputs an easy-to-understand summary for non-lawyers and stakeholders.

## Features

- Partner names and agreement type
- Key terms: scope, exclusivity, territory, duration
- Financial terms: revenue share, fees, payment schedules
- Obligations: what each party must do
- Termination conditions
- IP and data rights
- Liability and indemnification clauses
- What to watch out for (risk flags)
- Plain-English summary (executive-style)

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
