## App Concept

Generate A/B test hypotheses and experimental designs for conversion rate optimization. Input: current conversion rate, traffic volume, problem statement, element to test (CTA, headline, layout, image, price). Output: hypothesis statement, test design (control vs variant), success metrics, sample size calculator, statistical significance notes, and recommended test duration.
