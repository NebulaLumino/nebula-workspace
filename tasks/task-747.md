## App Concept

Generate project charter documents for project kickoffs. Input: project name, project sponsor, business case, key stakeholders, budget range, timeline, success criteria, risks known. Output: complete project charter with executive summary, scope statement, objectives and key results, stakeholder matrix, risk register summary, timeline milestones, and governance structure. Follows PMI/PMBOK format.
