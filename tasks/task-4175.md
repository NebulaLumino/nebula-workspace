## Description
Build an AI-powered OKR (Objectives & Key Results) tracking tool that helps teams set, align, and monitor goals. Features include AI-generated OKR suggestions, progress forecasting, risk detection, and weekly check-in facilitation.

## Requirements
- OKR creation wizard (company, team, individual levels)
- Key Result definition helper with measurable metrics
- Cascading OKRs across org hierarchy
- Weekly check-in prompts and progress updates
- AI-generated insights: at-risk OKRs, momentum scores, recommendations
- Milestone detection and auto-adjustment suggestions
- Integration with Jira, Asana, Linear, Notion
- Quarterly review and scoring reports
- 360° alignment view (CEO to IC)

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Jira / Asana / Linear API
- Vercel AI SDK
