## App Concept

Generate high-converting CTA (Call-to-Action) button text and surrounding microcopy. Input: offer type, desired action, platform (web, email, ad, social), urgency level, audience segment. Output: 8-12 CTA text options ranked by predicted conversion, surrounding microcopy phrases, button color suggestions (via copy), A/B test recommendations.

## Tech Stack

- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
