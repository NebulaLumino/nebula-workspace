## Overview
Build an AI-powered web app that automatically generates comprehensive README.md files and project documentation. Users paste their repository structure, describe their project in natural language, or provide an existing package.json/requirements.txt, and the AI generates a polished, well-structured README including badges, installation steps, usage examples, contributing guidelines, and license info.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Project type auto-detection (Node.js, Python, Rust, Go, etc.)
- File tree analysis for context-aware documentation
- AI-generated sections: Overview, Prerequisites, Installation, Usage, Configuration, API Reference, Contributing, License, FAQ
- Badge generation (CI status, version, license, npm downloads, Python PyPI)
- Code example generation with syntax highlighting
- Multi-language support (English, Chinese, Japanese, Spanish)
- One-click copy or download as markdown file
- Custom section reordering and editing
- Changelog / release history auto-generation
