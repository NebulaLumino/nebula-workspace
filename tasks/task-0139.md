## Research Summary

### Key Techniques (2026 State)

**LoRA (Low-Rank Adaptation)**
- Freeze base model weights, train only small adapter matrices
- 90-95% of full fine-tuning quality at 10-100x lower compute
- Recommended: rank 32, alpha 16, lr 1e-4, ~2000 steps

**QLoRA**
- LoRA + 4-bit quantization of base model
- Fine-tune 70B+ models on single GPU
- Unsloth: 2-5x faster, 80% less memory

**RLHF (Reinforcement Learning from Human Feedback)**
- Train reward model from human preferences
- Optimize policy with PPO against reward model
- DPO (Direct Preference Optimization): no separate reward model needed

**Instruction Tuning**
- SFT (Supervised Fine-Tuning) on curated instruction-response pairs
- Foundation for chat models (ChatGPT, Claude, etc.)

### Key Tools
- **Axolotl**: open-source fine-tuning toolkit
- **Unsloth**: fast LoRA fine-tuning, free tier
- **Tether QVAC**: BitNet models on consumer GPUs
- **Google Cloud**: G4 VMs with FP4 precision fine-tuning
