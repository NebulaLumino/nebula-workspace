## Overview
Build an AI-powered web app that generates correlation matrices and feature importance visualizations from a dataset. Users upload a CSV, the app calculates pairwise correlations and feature importance via random forest, and the AI explains key relationships and warns about multicollinearity.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts or visx for heatmap
- OpenAI SDK

## Features
- CSV upload with mixed numeric/categorical columns
- Pearson correlation matrix heatmap for numeric columns
- Point-biserial correlation for binary vs numeric
- Chi-square test for categorical vs categorical
- Feature importance from a simple random forest proxy (JavaScript-based)
- AI explanations: "X and Y are strongly correlated (r=0.87) — likely [reason]"
- Multicollinearity warnings (VIF-style flags)
- Highlight positive/negative/no correlation patterns
- Export as PNG heatmap image or CSV
