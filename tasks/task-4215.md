## Overview
Build an AI-powered web app that automatically generates comprehensive documentation for ETL pipelines. Users describe their pipeline steps (or paste pipeline config from Airflow, dbt, Fivetran, etc.), and the AI generates human-readable documentation: pipeline purpose, data flow diagram, schema documentation, data quality checks, and SLA notes.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK

## Features
- Paste pipeline config (YAML/JSON from dbt, Airflow DAG, Fivetran, etc.)
- Or describe pipeline steps in natural language
- Generates: pipeline overview, data flow diagram (ASCII art), table-level documentation
- Documents: source systems, destination schema, transformation logic
- Data quality check recommendations
- SLA/frequency notes (hourly, daily, weekly)
- Export as Markdown (dbt docs style, or Confluence)
- Auto-generates column-level lineage if schema is provided
- "Data contract" document generation for upstream/downstream teams
