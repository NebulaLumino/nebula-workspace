## Overview
Build an AI-powered web app that generates comprehensive test cases from function signatures, code snippets, or API endpoint specs. Users paste a function, class, or API definition, and the AI generates unit tests (Jest, Vitest, Pytest, Go test), integration tests, edge case tests, and property-based tests with expected inputs and outputs.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Language auto-detection: TypeScript, JavaScript, Python, Go, Rust, Java, Ruby
- Framework auto-detection: Jest, Vitest, Mocha, Pytest, Go testing, RSpec, JUnit
- Unit test generation with happy path + edge cases
- Mock/stub generation for dependencies
- Integration test generation for API endpoints
- Property-based / fuzz testing generation
- Error boundary and exception handling test coverage
- Coverage gap analysis: "your code handles X but you're not testing it"
- Copy as code or download as test file
- Batch mode: paste multiple functions → generate full test suite
- Test docstring generation with explanation of each case
