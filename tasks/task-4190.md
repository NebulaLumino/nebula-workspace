## Project
`ai-swot-from-web`

## Description
Build an AI-powered competitive intelligence tool that scrapes a company's website, press releases, and social media to auto-generate a structured SWOT analysis (Strengths, Weaknesses, Opportunities, Threats).

## Requirements
- URL input (company website, career pages, press room)
- Web scraping and content extraction
- AI SWOT extraction from textual signals
- Competitive comparison (vs 1-3 competitors)
- AI opportunity identification from market context
- Threat detection from news, reviews, social
- Interactive SWOT matrix editor
- Export to PowerPoint, PDF, Word
- Historical SWOT versioning
- Strategic recommendation engine from SWOT

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Playwright for scraping
- Vercel AI SDK
