## Description
Build an AI-powered real estate investment property analyzer. Input a property address or listing, and the app pulls market data, comparable sales, rental estimates, neighborhood analytics, and macroeconomic factors to generate an investment score, cap rate, cash-on-cash return, IRR projection, and AI-generated investment recommendation.

## Requirements
- Property address lookup (Zillow / Redfin API patterns)
- Comparable sales analysis (Comps)
- Rental income estimation
- Expense breakdown (taxes, insurance, maintenance, PM)
- ROI projections (5/10/20 year)
- Cap rate & cash-on-cash return calculator
- Neighborhood scoring (schools, crime, growth trends)
- AI investment narrative & recommendation
- PDF report generation

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Zillow API / RapidAPI Real Estate
- OpenAI SDK
- Chart.js / Recharts
