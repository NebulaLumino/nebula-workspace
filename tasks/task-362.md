## App Concept

A Next.js 16 web app that generates comprehensive SWOT analyses from company/market data. Users input a business or product and receive a structured SWOT matrix with supporting evidence and strategic recommendations.

## Features

- Company/product name + industry input
- AI-generated SWOT matrix (Strengths, Weaknesses, Opportunities, Threats)
- Each quadrant with 5-7 bullet points and evidence-based rationale
- Strategic implications for each quadrant
- Comparison mode: compare two companies side-by-side
- Export to PNG (visual SWOT grid) and PDF

## Tech Stack

- Next.js 16 (App Router)  
- Tailwind CSS  
- DeepSeek API (OpenAI-compatible)  
- TypeScript
