### Key Architecture Themes 2026

**MoE (Mixture of Experts):** Most flagship models use MoE — activates only a fraction of params per token → faster inference, lower compute cost. Llama 4 and Qwen3.5 both MoE.

**Multimodality:** Vision-language is now standard even in smaller models. Gemma 3, Mistral Small 4, Qwen3.5 all support image + text.

**Massive Context Windows:** Llama 4 Scout supports 10M token context. Most models offer 128K–262K. Crucial for agentic workflows and RAG.

**Agentic Focus:** Llama 4 and Mistral explicitly optimize for planning, tool use, and multi-step reasoning.

### Deployment Options

**Local (on-device):**
- Gemma 3 27B: fits on 1 consumer GPU
- Gemma 3 12B/4B: laptop-capable
- Ollama: easy local inference for any GGUF-format model
- LM Studio: GUI for local models

**Cloud / API:**
- Groq: fastest inference for open models (Llama, Gemma via API)
- Together AI, Anyscale, Fireworks: multi-model API access
- HuggingFace Inference Endpoints
