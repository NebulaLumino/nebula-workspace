## Description
An AI-powered tool for anger management that helps users identify anger triggers, practice emotional regulation techniques in real-time, and build long-term anger management skills. Includes breathing exercises, cognitive reframing, and a safe space to log anger episodes.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User logs an anger episode with context, trigger, and intensity
2. AI provides immediate de-escalation techniques (breathing, grounding)
3. AI analyzes the episode to identify patterns
4. AI generates personalized coping strategies for recurring triggers
5. Long-term progress tracking with insights
