## Overview
Build an AI-powered web app that generates polished CLI tools from a description of what the tool should do. Users describe the tool in natural language (e.g., "a CLI that converts CSV to JSON, supports filtering columns, and can gzip the output"), and the AI generates a complete CLI application with argument parsing, help text, shell completions, and man pages in the target language/framework.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript (Node.js / Click / Commander.js or Rust/Go for output)
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Natural language CLI description → full source code
- Framework selection: Click (Python), Commander.js (Node.js), Cobra (Go), Clap (Rust)
- Argument/flag generation with type inference (string, int, bool, enum)
- Auto-generated help text with examples
- Shell completions: bash, zsh, fish, PowerShell
- Man page (troff/groff) generation
- Interactive mode support (readline / inquirer)
- Colorized terminal output support
- Config file support (~/.toolrc, YAML/JSON)
- Cross-platform path handling (Windows, Unix)
- Test generation for the CLI (behavior-driven CLI tests)
