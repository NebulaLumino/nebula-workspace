## App Concept

Generate 30-day social media content calendars with post ideas, captions, and scheduling recommendations. Input: brand name, industry, platforms (Instagram, TikTok, LinkedIn, X, Facebook), posting frequency, campaign themes, holiday calendar. Output: daily/weekly posting schedule with post type, caption, hashtags, posting time recommendations, and engagement tips for each post.

## Tech Stack

- Next.js 16 (App Router), Tailwind CSS, DeepSeek API, TypeScript
