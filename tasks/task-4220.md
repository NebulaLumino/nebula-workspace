## Overview
Build an AI-powered log analysis tool. Users paste or upload a set of log lines (from any format: syslog, JSON, Apache, K8s, etc.), and the app uses AI to correlate events by timestamp, identify patterns, surface anomalies, and generate a root cause analysis report in plain English.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK

## Features
- Paste log text or upload .log/.txt file
- Auto-detects log format (syslog, JSON, Apache, k8s)
- Groups/correlation by time window
- Identifies: error patterns, repeated exceptions, spike events
- AI generates: "What happened: [summary], Likely cause: [X], Fix: [Y]"
- Severity classification per event (CRITICAL, ERROR, WARN, INFO)
- Timeline view: events plotted on a horizontal timeline
- Keyword extraction: top error messages, most frequent IPs/IDs
- Export root cause report as Markdown
