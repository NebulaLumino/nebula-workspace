## Description
Build an AI-powered peer-to-peer lending risk score calculator. Borrowers input financial data to receive a risk-adjusted credit score; lenders input loan listings to assess default probability. The AI analyzes alternative data (social, employment, spending patterns) to generate a more nuanced credit assessment than traditional FICO scores.

## Requirements
- Borrower profile input form
- AI creditworthiness scoring (alternative data)
- Loan listing analyzer (for lenders)
- Default probability model
- Expected return calculator (adjusted for risk)
- Platform comparison (Prosper, LendingClub, Funding Circle)
- Loan performance analytics
- Portfolio risk diversification suggestions
- P2P lending ROI tracker
