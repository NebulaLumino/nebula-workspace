## Research Summary

### What Are Prompt Libraries?
Collections of reusable, tested prompts for specific tasks — templates, chain-of-thought patterns, few-shot examples.

### Key Resources

**PromptBase** (promptbase.com)
- Marketplace for high-quality prompts
- DALL-E, Midjourney, ChatGPT, Stable Diffusion prompts

**SystemPrompts.io**
- Curated collection of system prompts
- Role-playing, coding, creative writing

**LangChain / LlamaIndex Prompt Hub**
- Structured prompt templates for RAG, agents, chain-of-thought

**OpenAI Cookbook**
- Official prompt engineering examples
- Few-shot learning, chain-of-thought, multi-step reasoning

### Key Patterns

**Chain-of-Thought (CoT)**
- "Let's think step by step" → better reasoning
- Self-consistency: multiple reasoning paths + vote

**Few-Shot Prompting**
- Provide 3-5 examples in the prompt
- Dramatically improves output quality

**Prompt Chaining**
- Break complex tasks into sequential prompts
- Output of one prompt feeds into the next

**Role Prompting**
- "You are an expert cardiologist..." → better domain responses
