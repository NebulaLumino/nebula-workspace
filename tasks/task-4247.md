## Overview
Build an AI-powered web app that scans code repositories for license compliance and missing copyright headers. Users upload a repo or paste a file tree, and the AI detects which open source licenses are in use, identifies missing or incorrect license headers, flags incompatible license combinations, and generates the correct copyright headers for the project's chosen license.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Upload repo / paste file list → license audit report
- Auto-detect license from LICENSE file (SPDX identifiers)
- Detect licenses from package manager files (package.json deps, requirements.txt)
- Missing copyright header detection per file
- Incorrect or outdated copyright year detection
- Incompatible license combination warnings (e.g., GPL + proprietary in same repo)
- Generate correct copyright headers for: MIT, Apache 2.0, GPL v2/v3, BSD, ISC, MPL, Unlicense
- Support for multiple file types: JS/TS, Python, Go, Rust, Java, C/C++, Ruby, Shell
- Custom header template builder
- License compatibility matrix (permissive vs copyleft)
- Contributor License Agreement (CLA) detection
- Dependency license summary (all transitive deps' licenses)
- Export report as JSON/Markdown
- SPDX license expression parser and validator
- AGPL/SSPL detection for database software
