## Research Findings — Multimodal AI 2026

### Leading Multimodal Models

| Model | Provider | Key Strengths |
|-------|----------|--------------|
| **GPT-4o** | OpenAI | Text + image + audio unified; fine-tuned medical versions; powers ChatGPT voice mode |
| **Gemini 3 Flash** | Google | "Agentic Vision" — actively investigates images with step-by-step reasoning; screen/camera share; video up to 90 min |
| **Gemini 2.5 Pro** | Google | Long PDF vision, chart extraction, bounding box object detection |
| **Llama 3.2 Vision** | Meta | 11B / 90B; excels at documents, charts, diagrams; open-weight |
| **Qwen2.5-VL** | Alibaba | Strong multilingual + vision; open-source; gaining traction |

### What Multimodal AI Can Do (2026)

**Image Understanding:**
- Describe, caption, and reason over any image
- Read text in images (receipts, docs, screenshots, handwritten notes)
- Chart/diagram comprehension and data extraction
- Medical imaging (specialized fine-tuned versions)
- Object detection with bounding box coordinates

**Visual Agents:**
- Gemini: "Agentic Vision" — formulates a plan to zoom/inspect an image step-by-step
- Observes a video (e.g., error on screen) → executes logic to resolve it
- Can see user's screen or camera in real-time → act on what it sees

**Document Understanding:**
- Native PDF + image processing (Gemini: documents up to thousands of pages)
- Extract structured data from web pages
- Screenshot → generate code, UI descriptions, accessibility text

**Video:**
- Gemini: summarizes and transcribes videos up to 90 minutes
- Frame-level understanding for film, surveillance, education

### The Shift: From "Describe" to "Act"

Early multimodal = "describe what's in this image"
2026 multimodal = "look at this, understand it, and do something about it"

**Agentic Vision** (Gemini's term) is the key shift: the model reasons over what it sees and takes actions — not just answering questions.

### Open Source Options
