## Description
An AI-powered tool that helps users with OCD practice Exposure and Response Prevention (ERP) by checking compulsions and generating rational responses. Users describe obsessions and compulsions, and the AI provides evidence-based reframing, probability analysis, and ERP-aligned challenge scripts.

## Technical
- **Framework:** Next.js 15 (App Router)
- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User describes an obsession or compulsive urge
2. AI provides ERP-aligned rational response and evidence-based challenge
3. AI generates probability-based analysis of feared outcomes
4. User rates anxiety before and after the response
5. AI tracks patterns and suggests exposure exercises
