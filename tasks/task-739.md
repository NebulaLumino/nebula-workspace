## App Concept

Generate Reddit posts optimized for subreddit engagement. Input: subreddit name and rules, post type (question, discussion, announcement, PSA), topic, target audience, karma level of subreddit, posting time preference. Output: post title with headline formula, post body with proper formatting, suggested flair, and engagement-boosting strategies. Includes comment reply templates for common responses.

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
