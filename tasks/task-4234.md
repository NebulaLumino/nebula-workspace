## Overview
Build an AI-powered web app that generates beautiful, structured changelogs and release notes from git commit history, GitHub issues, and PRs. Users connect a GitHub repo or paste commit logs, and the AI groups changes into categories (Added, Changed, Deprecated, Removed, Fixed, Security), detects breaking changes, and formats output in Keep a Changelog, GitHub Releases, or custom formats.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- GitHub repository integration (repo link or PAT token)
- Paste raw commit log or git log --pretty output
- AI-powered categorization (feat, fix, docs, refactor, perf, test, chore)
- Automatic breaking change detection (from commit messages, API diffs)
- Multiple output formats: Keep a Changelog, GitHub Flavored Markdown, JSON, YAML
- Version numbering suggestion (semver bump based on changes)
- Diff link generation for each change item
- Contributor acknowledgement extraction (from commit authors)
- One-click publish to GitHub Releases
- Custom branding: add logo, brand colors, company name
- Preview pane with live markdown rendering
