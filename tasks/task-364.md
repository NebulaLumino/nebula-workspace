## App Concept

A Next.js 16 web app that generates comprehensive competitive intelligence reports. Users input their company and competitors; the AI produces a detailed report covering market positioning, product comparison, pricing analysis, strengths, weaknesses, and strategic recommendations.

## Features

- Company + up to 5 competitors input
- AI-generated full competitive landscape report
- Product feature comparison matrix
- Pricing strategy comparison
- Market positioning map (2D perceptual map)
- Win/loss analysis for each competitor
- Battle cards for top 3 competitive threats
- Strategic recommendations and differentiation opportunities
- PDF export with professional formatting

## Tech Stack

- Next.js 16 (App Router)  
- Tailwind CSS  
- DeepSeek API (OpenAI-compatible)  
- TypeScript
