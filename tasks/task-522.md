## App Concept

Generate catchy, memorable promo codes for any campaign. Input: campaign type (flash sale, loyalty reward, new customer, holiday), discount type (%, $ off, BOGO), and number of variants needed. Output: a list of branded promo codes with suggested expiry, usage limits, and a printable/screenshot-ready card design.

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
