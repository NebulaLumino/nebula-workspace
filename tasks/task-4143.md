## Description
Build an AI-powered NFT valuation and rarity checker. Upload or input an NFT collection, and the app uses on-chain data, floor prices, recent sales, trait rarity analysis, and AI models to estimate fair value. Features include portfolio tracking, price history charts, and rarity scorecards.

## Requirements
- Multi-chain NFT support (Ethereum, Solana, Polygon, Base)
- Trait rarity scoring algorithm
- Floor price tracking
- Sales comparables analysis
- Portfolio tracker with P&L
- AI-powered valuation model
- Royalty fee analysis
- Whitelist/blue-chip score ranking
