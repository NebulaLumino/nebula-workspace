## Description
An AI-powered recovery tool that helps users track progress through addiction recovery programs (12-step, SMART Recovery, etc.), logs daily check-ins, identifies triggers, and connects users with AI coaching between human sponsor meetings. Includes crisis resource integration.

## Technical

- **AI SDK:** OpenAI (DeepSeek compatible)
- **Deployment:** Vercel

## Workflow
1. User selects recovery program type and current step
2. AI helps user work through step exercises and reflections
3. User logs daily check-ins, triggers, and milestones
4. AI generates trigger pattern reports and coping strategies
5. Includes crisis hotline and resource links
