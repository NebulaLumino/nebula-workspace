## Description
Build an AI-powered email triage tool that automatically categorizes, prioritizes, and summarizes incoming emails. Features include urgency scoring, smart threading, draft response generation, and scheduled send optimization.

## Requirements
- Gmail/Outlook API integration
- Email classification (urgent, action-required, FYI, newsletter, noise)
- Urgency scoring based on sender, content, keywords, time sensitivity
- Smart inbox segmentation (Primary, Action, Wait, Archive)
- AI-generated summaries for long threads
- One-click draft replies
- Unsubscribe helper and newsletter grouping
- Do Not Disturb scheduling

## Tech Stack

- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Gmail API / Microsoft Graph API
- Vercel AI SDK
