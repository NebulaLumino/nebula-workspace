## App Concept

Generate available domain name suggestions. Input: brand/product name, industry, target audience, preferred domain extension (.com, .io, .co, .ai), naming style (short, descriptive, invented, keyword-rich). Output: 30 domain name options with .com/.io/.co availability indicators, suggested alternatives if first choice taken, and acquisition difficulty rating (easy, moderate, premium).

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
