`ai-sop-writer`

## Description
Build an AI-powered SOP writer that generates comprehensive standard operating procedures from natural language process descriptions, workflow inputs, or existing scattered documentation.

## Requirements
- Process description input (text, audio transcript, or document)
- Step-by-step SOP structure generation (purpose, scope, steps, responsibilities)
- Decision point identification and branching logic
- Safety/warning callout insertion
- Tools/equipment checklist generation
- Quality control checkpoint insertion
- Estimated time per step
- RACI assignment auto-fill
- Version history and change log
- Export to Word, PDF, Confluence, Notion
