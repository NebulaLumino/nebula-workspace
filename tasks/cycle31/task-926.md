## Concept & Vision
A tool for Web3 users, Soulbound Token creators, and DID (Decentralized Identity) enthusiasts to craft comprehensive on-chain identity profiles. Generates identity attestations, credential portfolios, reputation narratives, and social proof documentation. Feels like having a DID architect building your complete digital identity.

## Specific Features
- DID/ENS identity setup guidance
- Credential type selection (work history/achievements/skills/ affiliations)
- Attestation request generation (for verifiers)
- Reputation narrative creation
- Skill endorsement templates
- Credential verification request flow
- Privacy controls (what to share vs. hide)
- Cross-platform identity linking
- Soulbound token metadata generation
- Identity recovery plan

## Input Fields
- Wallet address or ENS name
- Identity purpose (professional/networking/anonymous/developer)
- Credentials to include (work/education/achievements)
- Privacy preferences (full public / selective reveal / anonymous)
- Platform compatibility (Ethereum/Polygon/Arbitrum/Solana)

## Output
Complete DID profile package:
- Identity overview and bio
- Credential portfolio with descriptions
- Attestation request templates
- Verification flow guide
- Privacy configuration recommendations
- Cross-platform linking guide
- Recovery and backup plan
- Soulbound token metadata specs

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, JSON/credential export.
