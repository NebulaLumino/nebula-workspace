## Concept & Vision
A tool for keynote speakers, TED-style presenters, and corporate storytellers to craft compelling presentations designed for holographic or volumetric display formats. Generates full speaker scripts, stage direction notes, visual element descriptions, and audience interaction prompts optimized for next-generation display technology. Feels like having a future-forward presentation coach.

## Specific Features
- Holographic stage direction notation (depth planes, object placement)
- Volumetric audience perspective management
- 3D data visualization descriptions for volumetric screens
- Speaker script with timing markers
- Stage blocking and movement notes
- Ambient lighting and atmosphere descriptions
- Interactive polling and audience response integration
- Live demo cue system for holographic object reveals
- Q&A session structure for holographic format
- Post-presentation 3D assets summary

## Input Fields
- Presentation topic and core message
- Audience type and size (virtual/physical/hybrid)
- Presentation duration (10/20/45/60 min)
- Holographic format (pepper's ghost/holographic theater/volumetric/AR overlay)
- Technical constraints (stage size, projector specs)
- Speaker experience level

## Output
Holographic presentation package:
- Full speaker script with timing
- Stage direction and blocking notes
- 3D asset descriptions and specs
- Lighting and atmosphere guide
- Audience interaction plan
- Technical rider for AV team
- Rehearsal notes and pacing guide

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, PDF/JSON export.
