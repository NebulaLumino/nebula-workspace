## Concept & Vision
A tool for museum curators, tourism boards, and virtual experience designers to craft immersive audio/written tour narratives for physical or virtual spaces. Generates route descriptions, emotional beats, educational talking points, and sensory immersion cues. Feels like having a veteran museum docent and experience designer write your entire tour guide script.

## Specific Features
- Tour type (museum/historical site/nature/virtual space/corporate experience)
- Duration and pacing (30 min/1h/2h/multiday)
- Audience age and knowledge level
- Route stop generation with descriptions
- Audio tour script with timing markers
- Visual and environmental cue descriptions
- Interactive element integration (touchscreens, AR triggers, QR codes)
- Emotional arc construction (build-up → climax → reflective close)
- Accessibility accommodations per stop
- Quiz and engagement activity generation

## Input Fields
- Venue type and name
- Target audience demographics
- Tour duration preference
- Educational vs. entertainment balance
- Physical accessibility requirements
- Key highlights to feature
- Tone (scholarly/accessible/dramatic/playful)

## Output
Complete immersive tour package:
- Route map with 5-20 stops
- Full tour script per stop (audio-ready)
- Environmental description notes
- Interactive element placements
- Emotional arc timeline
- Accessibility accommodation checklist
- Quiz/activity bank per stop
- Staff briefing notes

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, PDF/audio script export.
