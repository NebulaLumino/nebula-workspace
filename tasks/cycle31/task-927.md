## Concept & Vision
A tool for metaverse developers, virtual world creators, and gaming economists to create compelling descriptions for virtual goods and digital assets. Generates item narratives, stat sheets, lore connections, and marketplace listings for virtual fashion, weapons, land, and collectibles. Feels like having a virtual economy designer write all your item descriptions.

## Specific Features
- Item category (clothing/accessory/weapon/land/vehicle/pet/consumable)
- Rarity and scarcity tiering
- In-world lore integration
- Stat and attribute descriptions
- Marketplace listing optimization (SEO for virtual goods)
- Bundle and bundle pricing strategy
- Trade restrictions and region locking
- Seasonal/event variant guidance
- Upgrade path descriptions
- Economic balance recommendations

## Input Fields
- Item name and category
- Rarity tier
- Virtual world or game context
- Lore universe (original IP or existing IP)
- Target player persona (completionist/whale/casual/trader)
- Price point or value range

## Output
Complete virtual goods package:
- Item display name with subtitle
- Full description (lore + mechanical function)
- Stat sheet with visual formatting
- Rarity tier description
- Marketplace listing copy (title, tags, description)
- Bundle strategy recommendations
- Trade restrictions spec
- Upgrade path narrative
- Economic balance notes

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, JSON/CSV export.
