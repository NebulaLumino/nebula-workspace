## Concept & Vision
A tool for gaming guild founders, esports organizations, and Web3 gaming guilds to create comprehensive guild charters and operational playbooks. Generates leadership structures, recruitment criteria, progression systems, and activity schedules. Feels like having a guild operations lead write your entire founding document.

## Specific Features
- Guild mission and values development
- Leadership structure (GM, officers, council roles)
- Membership tiers and progression path
- Recruitment criteria and application process
- Activity schedule (weekly raids, tournaments, social events)
- Loot/distribution rules (for P2E guilds)
- Contribution tracking system
- Alliance and diplomacy guidelines
- Code of conduct and disciplinary procedures
- Guild recruitment announcement copy

## Input Fields
- Game or games supported
- Guild type (casual/hardcore/competitive/P2E/mixed)
- Target size (casual 50 / medium 200 / large 1000+)
- Play-to-earn integration (yes/no/partial)
- Leadership team size

## Output
Complete guild charter:
- Mission and values statement
- Org chart and role descriptions
- Membership tier descriptions
- Application and onboarding process
- Weekly activity calendar
- Loot/distribution policy
- Contribution tracking system
- Code of conduct
- Disciplinary procedure guide
- Recruitment announcement template

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, printable charter format.
