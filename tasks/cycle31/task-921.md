## Concept & Vision
A tool for crypto artists, NFT project founders, and Web3 marketers to craft compelling descriptions, lore, and metadata for NFT collections. Generates collection narratives, individual trait descriptions, rarity tier copy, and marketplace listings. Feels like having a Web3-native creative director building your entire collection's story.

## Specific Features
- Collection concept and theme development
- Trait type and attribute naming (visual, narrative, utility)
- Individual NFT descriptions with personality
- Rarity tier copywriting (Common/Rare/Epic/Legendary)
- Lore integration across the collection
- Marketplace listing copy (OpenSea, Rarible, etc.)
- Whitepaper executive summary
- Community engagement hooks
- Collection roadmap narrative
- Social media announcement copy

## Input Fields
- Collection name and concept
- Total supply and trait count
- Art style reference
- Target community (gamers/collectors/art lovers/defi users)
- Utility features (staking, governance, game integration)
- Rarity distribution desired

## Output
Complete NFT collection package:
- Collection overview and tagline
- Full trait taxonomy with naming conventions
- Sample descriptions per rarity tier (10-20 examples)
- Collection lore narrative
- Marketplace listing copy
- Social media announcement templates
- Community FAQ
- Roadmap narrative

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, JSON metadata export.
