## Concept & Vision
A tool for Web3 founders, token economists, and protocol designers to model and document token economics systems. Generates token distribution tables, emission schedules, vesting curves, and sustainability analyses. Feels like having a tokenomics PhD building your economic model.

## Specific Features
- Token allocation across stakeholders (team/investors/community/treasury/ecology)
- Vesting schedule generation (cliff, linear, backend-loaded)
- Emission rate modeling (inflation/deflationary)
- Staking reward structure design
- Utility function specification (governance/staking/fee burn/discount)
- Token burn mechanisms
- Supply cap and inflation schedule
- Economic sustainability projections (year 1-5)
- Scenario modeling (bear/bull/neutral)
- Tokenomics one-pager generation

## Input Fields
- Token name and ticker
- Total supply
- Allocation percentages by category
- Vesting periods by stakeholder group
- Emission schedule preference
- Utility functions needed
- Target token price or market cap goal
- Protocol revenue model

## Output
Complete tokenomics model:
- Allocation table and pie chart description
- Vesting timeline table
- Emission schedule (monthly for years 1-5)
- Utility function specifications
- Staking reward model with APY estimates
- Burn mechanism design
- Economic sustainability analysis
- One-pager summary
- Presentation-ready charts (text format)

## Tech Stack
Next.js 16, TypeScript, Tailwind CSS, DeepSeek API, CSV/JSON export.
