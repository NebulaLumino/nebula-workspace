## App Concept

Paste a landing page URL or HTML and get a full UX/copy teardown powered by AI. Input: URL or page content. Output: section-by-section scoring (headline, CTA, social proof, trust signals), conversion killers identified, heatmap-style predictions, and specific rewrite suggestions for every weak element.

## Tech Stack

- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
