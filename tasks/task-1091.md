# Brand Style Guide & Visual Identity Generator

## Overview
An AI-powered tool that generates comprehensive brand style guides including color palettes, typography systems, logo usage rules, voice & tone guidelines, and visual identity components.

## Technical Implementation
- **Framework:** Next.js 16 (App Router)
- **AI:** OpenAI SDK (compatible with DeepSeek)
- **API:** DeepSeek (sk-48987c1a1dc246ecb1b52a01647e8b16)
- **Endpoint:** https://api.deepseek.com/v1
- **Model:** deepseek-chat

## Features
1. Brand name & industry input
2. Tone/personality selection (professional, playful, luxury, minimal, bold)
3. Target audience description
4. Comprehensive style guide output in markdown

## Generated Output Sections
- Color Palette (primary, secondary, accent with hex codes)
- Typography System (headings, body, monospace)
- Logo Usage Rules & Clear Space
- Voice & Tone Guidelines
- Imagery Style & Photography Direction
- Iconography Style
- Pattern & Texture Direction
- Do's and Don'ts
