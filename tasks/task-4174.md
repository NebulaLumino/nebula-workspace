## Description
Build an AI-powered onboarding tool that generates personalized checklists and training plans for new hires based on their role, department, seniority, and company. Includes progress tracking, task assignments, and manager dashboards.

## Requirements
- Role/department template library
- AI-personalized task list generation based on job profile
- Day-1 through Day-90 milestone planning
- Tool access checklist (software, accounts, permissions)
- Training resource recommendations (courses, docs, videos)
- Buddy/mentor assignment suggestions
- Progress tracking dashboard for managers
- New hire feedback loop
- Completion certificates

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Notion / HRIS API integration
- Vercel AI SDK
