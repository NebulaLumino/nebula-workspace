## App Concept

Generate compelling 3-paragraph brand positioning statements. Input: brand name, target audience, category, key benefit, competitor, and differentiator. Output: structured positioning that can be used across all marketing materials.

## Features

- Multi-field input form (brand, audience, category, benefit, competitor, differentiator)
- AI generates 3-paragraph positioning: lamp post, competitor comparison, brand promise
- Copy-to-clipboard + regenerate with one click
- Tone selector (bold, subtle, luxury, approachable)

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
