## App Concept

Generate powerful company mission, vision, and values statements. Input: company name, industry, founding story or origin idea, target customer, core problem solved, long-term aspiration, team values. Output: mission statement, vision statement, core values list (5-7 values with names and descriptions), and a brand manifesto paragraph. Multiple style variations (aspirational, operational, customer-focused, change-oriented).
