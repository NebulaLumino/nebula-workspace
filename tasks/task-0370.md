## App Concept

A Next.js 16 web app that generates professional press/media kits for individuals and companies. Includes bio, stats, brand guidelines, past coverage, and downloadable-ready sections.

## Features

- Personal/company profile: name, role, company, bio
- Stats section: follower counts, metrics, achievements
- Brand guidelines: colors, fonts, logo usage rules
- Headshot/brand image placeholder references
- Past media coverage: list format with publication names
- Speaking topics and availability
- Contact information
- Download/copy-ready sections

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
