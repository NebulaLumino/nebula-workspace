## Description
Build an AI-powered expense categorization tool that imports invoices and receipts, auto-categorizes expenses using natural language understanding, detects anomalies, and generates financial summaries.

## Requirements
- Receipt/invoice image upload (OCR via AI)
- Bank transaction CSV import
- AI-powered category suggestion (not rigid rules)
- Custom category training from user corrections
- Expense report generation (weekly, monthly, project-based)
- Vendor/destination analysis
- Anomaly detection (unusual spend, duplicate entries)
- Tax-deductible flagging
- Multi-currency support with FX conversion
- Integration with QuickBooks, Xero, FreshBooks

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Tesseract.js or AI OCR
- Vercel AI SDK
