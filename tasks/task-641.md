## App Concept

Generate personalized influencer outreach emails. Input: influencer name, platform, follower count, content niche, brand/product being promoted, campaign type (sponsored post, affiliate, ambassadorship, product seeding), compensation offer. Output: personalized outreach email with subject line options, intro hook, collaboration pitch, value proposition for the influencer, CTA, and follow-up sequence.

- DeepSeek API (OpenAI-compatible)
