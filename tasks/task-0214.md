## Research Findings — AI Video Editing Agents 2026

### The Landscape

| Tool | Type | What It Does |
|---|---|---|
| **Runway Gen-3** | Generation + Editing | Text-to-video, image-to-video, video-to-video |
| **Kling AI** | Generation | High-quality 1080p video from text/image |
| **Pika 2.0** | Editing | Edit specific parts of video (Pikachu effects) |
| **Sora** | Generation | OpenAI's long-form video generator |
| **CapCut AI** | Editing | Auto-cuts, captions, transitions |
| **Descript** | Editing | Edit video by editing transcript |
| **Adobe Firefly** | Generation + Editing | Premiere Pro integration |

### How AI Video Agents Work

1. **Text-to-Video**: Prompt → scene generation → temporal consistency
2. **Video Editing**: Transcript-based cuts, B-roll insertion, color grading
3. **Video-to-Video**: Style transfer, face swap, object replacement
4. **Captioning**: Whisper transcription + AI caption styling

### Practical Workflows

**Content Creator Pipeline:**
```
Script → Runway/Kling (generate B-roll) → CapCut (assemble + captions) → YouTube
```

**Post-Production Pipeline:**
```
Raw footage → Descript (auto-cut) → Adobe Firefly (enhance) → Final cut
```

### Key Capabilities

| Feature | Status 2026 |
|---|---|
| Text-to-video | ✅ Mature (Runway, Kling, Sora) |
| Auto-captions | ✅ Excellent (CapCut, Descript) |
| Background removal | ✅ Excellent |
| Face swap | ✅ Good |
| Video extension | ✅ Good (Runway Extend) |
| Full movie generation | ⚠️ Still limited (~5-10s clips) |

### Limitations
- Video generation still produces ~5-20s clips max
- Temporal consistency (character/object persistence) is hard
- Motion quality still inferior to real footage
- Cost per minute of generated video is significant
