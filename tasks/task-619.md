## App Concept

Generate affiliate program commission structures and program terms. Input: product type, average order value, customer LTV, competitor commission rates, budget constraints, program goal (acquisition, retention, brand awareness). Output: tiered commission structure, commission rates by tier, cookie duration, payment terms, promo code discounts, and program launch checklist.
