## Description
Build an AI-powered precious metals and commodities price prediction tool. Track gold, silver, platinum, oil, natural gas, and agricultural commodities. Uses time-series forecasting (LSTM/Prophet), macro-economic indicator correlation, Fed policy impact analysis, and AI to generate short and medium-term price predictions with confidence intervals.

## Requirements
- Real-time commodity price feeds
- Historical price charts (5D, 1M, 1Y, 5Y, All)
- AI price prediction model (LSTM or Prophet)
- Macro-economic indicator overlay (CPI, interest rates, USD index)
- Fed policy impact analysis
- Portfolio hedging recommendations
- Alert system for price targets
- Correlation matrix (gold vs stocks vs crypto)
- Weekly AI market outlook reports

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Alpha Vantage / FRED API
- Quandl / Yahoo Finance
- OpenAI SDK
- Recharts
- Python ML backend (FastAPI + Prophet/LSTM)
