## Overview
Build an AI-powered web app that transforms raw data into compelling data stories. Users paste or upload a dataset, and the AI analyzes it, identifies key patterns and trends, and generates a structured narrative report with charts, insights, and executive summary — turning numbers into a story with beginning, middle, and resolution.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- OpenAI SDK

## Features
- CSV upload or paste tabular data
- AI auto-selects the 3-5 most important metrics
- Generates structured report: Executive Summary → Key Findings → Supporting Charts → Recommendations
- Auto-inserts chart placeholders for each finding
- Tone selector: formal, conversational, executive brief, journalistic
- Export as Markdown or HTML
- Slide mode: one finding per slide
- Key quote/callout extraction from data
- Trend analysis: is metric going up, down, or flat?
