## Description
Build an AI-powered insurance claim denial appeal letter writer. Users input claim details, denial reason, policy information, and medical/documentation context. The AI generates a professionally structured appeal letter citing relevant laws (ERISA, state insurance codes), medical necessity arguments, and supporting documentation checklists.

## Requirements
- Claim denial input form
- Appeal letter generator (multi-type: health, auto, home, life)
- Law citation database (ERISA, state codes)
- Medical necessity argument builder
- Documentation checklist generator
- Appeal timeline tracker
- Success probability estimator
- Template customization by insurance type
- Appeal letter versioning
- PDF export for submission

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- OpenAI SDK (specialized fine-tuned model for legal language)
- PDF generation
- local law citation database (JSON)
- Form persistence (localStorage / Prisma)
