## Overview
Build an AI-powered dashboard for uploading time series data and detecting anomalies. Users upload a CSV with timestamp + value columns, and the app visualizes the series, highlights anomalies using statistical methods, and generates an AI explanation of what likely caused each anomaly.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- OpenAI SDK

## Features
- CSV upload with timestamp + numeric value columns
- Time series line chart with anomaly markers
- Anomaly detection: Z-score, IQR, rolling window deviation
- AI-generated explanation of each anomaly
- Filter by date range
- Download anomaly report as CSV
- Configurable sensitivity threshold
- Summary stats: mean, median, std dev, trend direction
