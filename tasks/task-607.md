## App Concept

Generate loyalty program structures and rule sets for e-commerce brands. Input: brand name, industry, average order value, customer lifetime value, budget per reward, program type (points, tiers, VIP). Output: program structure, point earning/redemption rules, tier benefits, terms & conditions, and launch timeline.
