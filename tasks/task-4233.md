## Overview
Build an AI-powered web app that analyzes code snippets, files, or pull request diffs and generates constructive code review comments. The AI acts as a senior engineer reviewer — identifying bugs, security issues, performance concerns, style inconsistencies, and offering refactoring suggestions with concrete code examples.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (compatible with DeepSeek)

## Features
- Paste code / diff / link GitHub PR → structured review comments
- Comment categories: Bug, Security, Performance, Style, Design, Testing, Documentation
- Severity levels: Blocker, Suggestion, Nitpick
- Inline code suggestions (before/after snippets)
- Explanation of why each issue matters
- Auto-assign reviewer persona (Backend expert, Security auditor, Performance profiler)
- Summary dashboard: issue count by category/severity
- "Approve with comments" vs "Request changes" summary
- Integrate with GitHub Actions workflow (PR comment via API)
- .github/CODEOWNERS aware review scoping
