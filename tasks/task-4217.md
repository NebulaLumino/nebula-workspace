## Overview
Build an AI-powered glossary builder for business metrics. Users list their KPI/metric names (from a spreadsheet, dashboard, or typed list), and the AI generates comprehensive metric definition cards: what it measures, how it's calculated, data source, owner, typical ranges, and common pitfalls.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK

## Features
- Paste or type a list of metric names
- Optional: paste dashboard screenshot description or BI tool export
- AI generates metric cards with: definition, formula, data source, owner team, update frequency
- Traffic light health indicators (is this metric stable/variable?)
- Links to related metrics (metric taxonomy)
- Tags: revenue, product, marketing, ops, financial
- Search and filter glossary
- Export as Markdown, CSV, or Notion-compatible format
- Version history for metric definition changes
- Metric review status: approved, draft, deprecated
