## Project
`ai-report-generator`

## Description
Build an AI-powered report generator that connects to business data sources (CSV, Excel, databases, APIs) and auto-generates formatted reports with narrative insights, charts, and executive summaries.

## Requirements
- Data source connectors (CSV, Excel, Google Sheets, PostgreSQL, Airtable)
- Natural language query interface (ask your data)
- Auto-chart selection based on data type
- Narrative insight generation (explain the "so what")
- Multiple report templates (executive summary, monthly, quarterly, KPI)
- Scheduled report generation and email delivery
- Export to PDF, PPTX, Google Docs
- Branding/customization (logo, colors, fonts)
- Interactive drill-down exploration

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Recharts / Tremor for charts
- Google Sheets API
- Vercel AI SDK
