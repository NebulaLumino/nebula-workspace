## Description
Build an AI-powered on-chain wallet risk scoring tool. Input any Ethereum/BNB/Solana wallet address and receive a comprehensive risk score based on interaction history, protocol exposure, token composition,MEV bot exposure, bridge risk, and suspicious activity patterns. Useful for DeFi investors, compliance teams, and security researchers.

## Requirements
- Multi-chain wallet analysis (Ethereum, BSC, Solana, Polygon, Arbitrum)
- Token/asset composition analysis
- Protocol interaction history
- Bridge usage risk assessment
- MEV bot exposure detection
- Phishing/hack association check
- AI-generated risk narrative
- Portfolio value estimation
- Risk score comparison to network average
