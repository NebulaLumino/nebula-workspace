## App Concept

A Next.js 16 web app that generates monthly email newsletters with compelling headline, intro paragraph, 3-4 content sections, and a strong CTA. Used by marketers and solopreneurs to stay in front of their audience.

## Features

- Company/person brand context
- Newsletter theme/month focus
- Headline generation (3 options)
- Opening hook paragraph
- 3-4 content sections: updates, insights, stories, tips
- Product/service spotlight section
- CTA: primary and secondary calls-to-action
- P.S. line generation
- Social share snippet

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
