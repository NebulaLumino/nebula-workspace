## Description
Build an AI-powered performance review tool that pulls data from project management tools, feedback systems, and peer inputs to generate structured, objective, and actionable performance reviews.

## Requirements
- Data source connectors (Jira, Asana, GitHub, Slack, Lattice, BambooHR)
- Goal achievement analysis against OKRs/KPIs
- Peer feedback aggregation (anonymized)
- Manager note input and narrative synthesis
- AI-generated review sections: Accomplishments, Areas for Growth, Goals
- Calibration support (normalize ratings across team)
- Development plan auto-generation
- Review draft comparison and iteration
- GDPR-compliant data handling
- Export to HRIS (Lattice, BambooHR, Workday)

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI SDK (GPT-4o)
- Jira / GitHub / Slack API
- Vercel AI SDK
