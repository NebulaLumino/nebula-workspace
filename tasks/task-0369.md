## App Concept

A Next.js 16 web app that generates investor-ready one-pagers — concise single-page investment summaries covering problem, solution, traction, team, and funding ask. Used by founders to pitch VCs, angels, and seed investors.

## Features

- Company snapshot: name, tagline, founded, HQ, stage
- Problem statement: quantified pain point
- Solution description: product overview, key differentiators
- Traction: revenue, customers, growth metrics
- Team: founder backgrounds, relevant expertise
- Market size: TAM/SAM/SOM
- The Ask: amount, use of funds, round size
- Contact/next steps section
- PDF-ready layout (print-optimized)

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- DeepSeek API (OpenAI-compatible)
- TypeScript
