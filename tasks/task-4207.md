## Overview
Build an AI-powered funnel analysis tool. Users input funnel stage data (stage name + user count at each stage), the app visualizes the funnel, identifies the biggest drop-off stages, and uses AI to generate actionable recommendations for improving conversion at each stage.

## Tech Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Recharts
- OpenAI SDK

## Features
- Funnel stage input: name + user count per stage
- Visual funnel chart with drop-off percentages between stages
- Highlights the worst-performing (highest drop-off) stages
- AI generates: why drop-off might be happening + specific recommendations
- Compare up to 3 funnels side-by-side (A/B test or time periods)
- Shows conversion rate and absolute drop per stage
- Export report as Markdown or CSV
- Pre-built templates: e-commerce checkout, SaaS signup, lead gen
